    第1章：资产评估概述5

    概念

    资产评估

    • 一般意义：估计和判断资产的价值
    • 专业服务：依据执业标准评定估算。
    • 法律规定：评估机构及其评估专业人员根据委托对不动产、动产、无形资产、企业价值、资产损失或者其他经济权益进行评定、估算，并出具评估报告的专业服务行为。

    评估主体

    评估机构及其评估专业人员（资产评估师/其他评估从业人员）。

    评估客体

    资产评估的对象（不动产、动产、无形资产、企业价值、资产损失或其他经济权益等）。

    资产基本特征：

    • 是特定主体拥有或控制的
    • 能带来未来经济利润
    • 资产是经济资源：有用和稀缺、能够用货币计量和反映

    特点：公正性、鉴证、咨询、专业

    1. 公正性：①技术保证：法定准则，公允的行为规范和业务规范 ②组织保证：无利害关系（独立性）
    2. 鉴证性：资产评估从事的是价值鉴证，而不是权属鉴证。
    3. 咨询性
    4. 专业性

    原则

    工作原则：独立、客观、公正

    也是职业道德要求。1. 独立、2. 客观、3. 公正

    经济技术原则：供求、最佳、替代、预期收益、贡献、时点

    1. 供求原则：体现了资产评估既需要考虑资产的购建成本，又需要考虑资产的效用。
    2. 最高最佳使用原则：法律许可、技术可能、财务可行。
    3. 替代原则：是指价格最低的同质商品对其他同质商品具有替代性，正确运用替代原则是资产评估公正性的重要保证。
    4. 预期收益原则：预期收益原则指资产的价值不在于过去的生产成本或销售价格，而是应当基于其对未来收益的预期加以决定。是资产评估专业人员判断资产价值的一个最基本的依据。
    5. 贡献原则：贡献原则主要是指某一资产或资产某一构成部分的价值，取决于它对所在资产组合或完整资产整体价值的贡献。贡献原则是预期收益原则在某种情况下的具体应用。
    6. 评估时点原则：评估时点原则也是对交易假设和公开市场假设的一个反映。

    地位

    1. 资产评估是市场经济不可或缺的专业服务行业。
    2. 资产评估是现代服务业的重要组成部分。 资产评估行业具备典型的现代服务业特征，一是服务主体具有专业性；二是服务对象具有高端性；三是服务行为具有公正性。
    3. 资产评估是我国经济体制改革逐步深化的重要专业支撑力量。

    作用

    1. 资产评估在促进市场资源优化配置中的引导作用：引导经济资源向价值最大化方向流动，防止“劣币驱逐良币”现象的产生
    2. 资产评估在服务资本市场发展中的保障作用：在推进上市公司并购重组、促进融资功能提升、提高资本市场信息披露质量方面发挥了至关重要的作用
    3. 资产评估在规范经济秩序中的促进作用：抑制了交易主体的非理性行为。
    4. 资产评估对维护各类资产权利主体的合法权益有积极作用：使交易各方能够在公开、公正、公平的前提下，实现资产最有可能实现的交换价值，达到帕累托最优状态
    5. 资产评估对维护公共利益和对外开放环境下的国家利益有重要作用。

    1. 劳动价值论

    理论发展过程：威廉配弟、亚当斯密、大卫李嘉图、马克思。劳动价值论认为价值实体是客观的，衡量价值的尺度也是客观的。因此劳动价值论又被称为客观价值论。

    商品两因素原理

    1. 商品是用来交换的劳动产品
    2. 商品包括使用价值和价值两个要素，是两个价值的统一
      1. 使用价值是商品的自然属性；是交换价值的物质承担者。
      2. 价值是一般人类劳动的凝结，是商品的社会属性，它构成商品交换的基础。交换价值是价值的表现形式，而价值是交换价值的基础，没有价值的东西也就没有交换价值。
    3. 商品使用价值与价值的联系
      1. 二者是统一的，互相依存、互为条件
      2. 二者又是对立的

    劳动二重性

    1. 具体劳动：创造了商品的使用价值
    2. 抽象劳动：创造了商品的价值
    3. 具体劳动和抽象劳动是同一劳动过程从不同角度去观察的两个方面。二者在时间上、空间上都是不可分割的。

    商品价值量的决定

    1. 社会必要劳动时间：不等于个别劳动时间
    2. 劳动生产率
    3. 商品价值量：
      1. 商品的价值量只能由生产商品的社会必要劳动时间决定。
      2. 确定商品价值量的社会必要劳动时间随着劳动生产率的变化而变化的。
      3. 商品价值量取决于生产商品的劳动时间，且与劳动时间成正比。
      4. 同一劳动在同样的时间内提供的价值量总是相同的，在社会必要劳动时间决定价值的经济中， 劳动生产率高的个别价值在还原为社会价值时，会还原为更高的价值。

    价值规律

    1. 价值规律的作用形式：价格围绕价值波动。 价值决定价格，价格反映价值。
    2. 价值量的决定因素：社会必要劳动时间
    3. 价值规律的重要作用：1. 自发调节。2. 刺激生产积极性。3. 优胜劣汰，导致生产者两极分化。

    对劳动价值论的认识

    1. 商品的价值量由社会必要劳动时间决定
    2. 商品是使用价值和价值的对立统一体，两者对立统一，不可分割。
    3. 商品的价值要转化为价格就必须依靠市场。
    4. 认清价值的本质是无差别的人类劳动，不能简单以市场价格代替价值。

    2. 效用价值论

    边际效用递减规律

    1. 效用：不具有客观性
    2. 边际效用
    3. 边际效用递减：边际效用是总效用的斜率

    消费者均衡与效用最大化：最后一元带来的边际效用相等

    商品价值

    1. 价值来源：效用价值论认为价值不是商品的内在属性，而是人的主观评价形成的一种心理范畴。价值应分为主观价值和客观价值。
      1. 主观价值：根源于有用性和稀缺性
      2. 客观价值：交换价值
    2. 价值量的确定
      1. 是商品的最小效用（边际效用）
      2. 边际效用的大小是由需求和供给关系决定的
      3. 有用性和稀缺性是决定物品的最终因素：有用程度表示（在极端情况下）边际效用预计可能达到的高度，稀缺性决定了具体情况下边际效用实际达到的水平。
      4. 主观评价决定均衡价格。
    3. 价格形成的基本规律：主观评价决定均衡价格。消费者均衡是研究如何把既定的收入分配在各种商品的购买中以获得最大的效用。也可以说，它是研究消费者如何在预算、价格、偏好等条件不变的情况下，实现自己买到的产品组合效用的最大化。

    对效用价值论的认识

    1. 价值产生于人们对物品效用的主观评价。
    2. 意义：开辟了从需求角度衡量价值的观点，可以用效用大小来衡量其价值，为均衡价值论提供了理论基础。
    3. 局限性
      1. 由于效用价值是主观的评价，运用效用价值对资产进行评估往往带有一定的主观性，导致评估出的价值与客观价值存在一定的差异。
      2. 边际效用价值论过于强调商品效用带给人的主观上的满足，忽略了交换和交换背后的社会经济关系， 过分夸大了效用的作用。

    • 需求关系的特殊情况
      • 替代效应：因某商品名义价格变化，而导致的消费者所购买的商品组合中，该商品与其他商品之间的替代。猪肉价格上涨了，牛肉价格价格不变，那么牛肉价格相对于猪肉价格就便宜了，在其他条件不变的情况下，消费者就可能增加牛肉的消费。

    4. 市场结构理论

    市场结构划分因素：行业进出难度、厂商数量、价格影响力、产品差别

    完全竞争市场：

    1. 概念
    2. 条件
      1. 有大量的买者和卖者（单一买者或单一卖者不能决定市场价格）。
      2. 每个厂商提供的都是完全同质的商品。 消费者无特定产品偏好，厂商不能区别定价，商品之间具有完全替代性，厂商会按照市场已经形成的 价格维持属于他们自己的市场份额。
      3. 各种资源能够自由流动，不存在任何阻碍和干扰。
      4. 信息畅通、完全（不会因信息不对称而相互欺诈）。
    3. 需求曲线：一条水平线。且厂商的平均收益曲线、边际收益曲线和需求曲线重合。（三线合一）
      1. 既定价格水平上，单个厂商所面临的产品需求量是无穷大的。
      2. 无论单个厂商如何改变销售量，均无法改变既定的市价。
      3. 如果厂商试图提高价格，销售将降为零。
      4. 形成新的均衡价格时，在完全竞争市场又会形成一条与该均衡价格对应的新的水平线
      5. 边际收益等于边际成本的原则实现利润最大化或亏损最小化目标。
    4. 短期均衡
      1. 在短期，厂商在生产规模不变情况下，可调整产量使边际收益等于短期边际成本，实现利润最大化。
      2. 如果处于亏损状态，厂商通过比较平均收益和平均可变成本决定是否继续生产。
    5. 长期均衡：在长期，厂商可调整全部生产要素使边际收益等于长期边际成本，达到利润最大化。
      1. 成本不变行业：行业的长期供给曲线是一条水平线。
      2. 成本递增行业：行业的长期供给曲线向右上方倾斜。
      3. 成本递减行业：行业的长期供给曲线向右下方倾斜。

    垄断竞争市场：

    1. 概念：垄断竞争市场又称为不完全竞争。
    2. 特点
    3. 条件：①行业中存在着大量的厂商，无法对市场形成控制。 ②生产有差别的同种产品，但产品之间具有替代性。 ③厂商生产规模较小，不存在进出障碍。
    4. 需求曲线：
      1. 向右下方倾斜，比较平坦
      2. 有两条需求曲线：主观需求和实际需求曲线，主观更平弹性更大。
    5. 供给曲线：包括垄断竞争市场在内的所有非完全竞争厂商都没有供给曲线，因为它们都不满足商品的价格与其供给量存在一一对应关系的条件。
    6. 短期均衡：
      1. 在短期，厂商仍然通过调整产量和价格使边际收益等于短期边际成本，实现利润的最大化；
      2. 如果处于亏损状态，厂商同样需要通过比较平均收益和平均可变成本来决定是否继续生产。
    7. 长期均衡
      1. 在长期，厂商可以通过调整生产规模使边际收益等于长期边际成本，追求利润的最大化。
      2. 由于厂商进出行业比较容易，长期均衡时垄断竞争厂商的利润一定为零。（因为只要存在利润就会有新的厂商进入到行业中）
      3. 由于垄断竞争厂商数量过多，且每个厂商的规模都过小，垄断竞争厂商在长期均衡时的产量会小于完全竞争厂商在长期均衡条件下的理想产量，这就使单个垄断竞争厂商存在未被利用的多余生产能力。

    寡头垄断市场

    1. 概念
    2. 特点：纯粹寡头市场产品没有差别，厂商的相互依存程度高。 差别寡头市场的产品则是有差别的，厂商的相互依存度低。
    3. 形成原因
      1. 追求规模经济效益，促使行业生产向大规模厂商集中；
      2. 少数厂商控制了行业基本生产资源的供给；
      3. 法律或政策的推动等。
    4. 分类：按照产品特征分类，可分为纯粹寡头行业和差别寡头行业。
    5. 典型模型
      1. 古诺：双头
      2. 伯特德：价格竞争
      3. 斯塔克伯格：产量领导型
      4. 价格领导型：1. 支配型、2. 成本最低型、3. 晴雨表型

    完全垄断市场

    1. 概念
    2. 条件
    3. 形成原因
      1. 竞争和规模经济要求
      2. 专利保护
      3. 政策法规
      4. 自然条件
    4. 需求曲线：市场即厂商的需求曲线，向右下方倾斜
    5. 短期决策：垄断厂商在生产规模不变情况下，通过调整产量和价格，使边际收益等于短期边际成本，达到利润最大化。垄断厂商按照上述原则调整产量与价格后的盈利情况，还需分析其平均成本状况。
    6. 长期决策：垄断厂商可以通过调整生产规模，使边际收益等于长期边际成本，实现最大的利润。其长期均衡的利润总大于短期均衡的利润。

    5. 有效市场理论

    理论形成：有效市场理论的最终形成及完善得益于尤金·法玛的卓越贡献。1965年，尤金·法玛在《金融分析家》杂志上发表《股票市场价格行为》一文，首次提出"有效市场"的概念。

    观点： - 有效市场假说认为，价格完全反映了全部所获得信息的证券市场是有效市场。 - 信息有效、投资者理性和市场理性的统一造就了有效的市场。

    前提条件 - 市场上的投资者都是理性的经济人，都以追求利益最大化为行动目标，投资人都力图利用所获信息谋取最高的利润。 - 与投资相关的信息都以随机方式进入市场，信息的发布各自独立。 - 市场对信息的反应迅速而准确，因而股票价格反映了市场的全部信息。 - 整个市场完全竞争，有大量投资者参与，大家都是价格的接受者。

    有效市场形态：弱式、半强式、强式

    有效市场假说检验 - 弱式：技术分析 - 股票价格时间序列分析 - 股票价格随机性分析 - 股票交易策略有效性 - 半强式：基本分析是否有用 - 事件研究法：事件引起股价快速反应，则假设成立 - 强式：内幕消息是否有用

    其它理论：威斯特和提尼克（West 和 Tinic）1975年在《金融分析家》杂志发表《论内外效率的差异》。 - 内在效率：反映其运行效率，指市场的组织机构和服务设施，能否使交易者以最短时间和最低成本达成交易。 - 外在效率：反映其资金配置效率，指资本市场价格能否及时迅速对市场信息作出反应。衡量一个证券市场是否具有外在效率： - 一要了解证券价格能否伴随信息变化自由变动； - 二要研究市场信息的披露状况，能否使与证券有关信息充分披露，让每个投资者能在相同时间获得等量等质的信息。

    第3章：资产评估法律制度与准则5

    法律体系

    目的

    概念：资产评估目的，实际就是资产评估业务对应的经济行为对资产评估结果的使用要求，或资产评估结论的具体用途。委托人计划实施的经济行为决定了评估目的。经济行为：企业改制、上市，合资、合作经营，股权转让，资产买卖、置换、出资，债转股，担保融 资，破产清算，保险赔偿，损失补偿，税收，司法诉讼，会计计量等。

    作用：评估目的直接或间接地决定和制约着资产评估的条件以及价值类型的选择。不同评估目的可能会对评估对象的确定、评估范围的界定、价值类型的选择以及潜在交易市场的确定等方面产生影响。 常见的资产评估目的 1. 转让：涉国与不涉国资，涉上市与不涉上市 2. 抵押质押：前后 3. 公司设立、增资：不得以劳务、信用、自然人姓名、商誉、特许经营权或者设定担保的财产等作价出资 4. 改制：有限责任公司变更为股份有限公司时，折合的实收股本总额不得高于公司净资产额。 5. 财务报告：合并对价分摊 、资产减值测试 、投资性房地产和金融工具等资产的公允价值计量 6. 税收：①核定税基 ②确定计税价格 ③关联交易转让定价等 7. 司法：司法评估事项是法院司法判决或执行程序的组成部分。诉讼协助评估属于当事人根据自己意愿决定的评估事项。人民法院应当在参考价确定后十日内启动财产变价程序。

    对象

    概念： - 《资产评估法》所表述的评估对象包括不动产、动产、无形资产、企业价值、资产损失或者其他经济权益。 - 《资产评估行业财政监督管理办法》所表述的评估对象包括单项资产、资产组合、企业价值、金融权益、资产损失或者其他经济权益。 - 评估对象可以是完整权利形态，即资产的所有权，也可以是资产部分权利，如租赁权、特许权、 经营权等。 - 资产权可以是物权、知识产权或债权。

    评估对象的确定：应当由委托人依据法律法规的规定和经济行为的要求提出，并在评估委托合同中明确约定。

    评估范围

    概念：是对评估对象所进行的详细描述，包括构成、物理及经济权益边界、约束条件等内容，是资产评估专业人员根据评估目的界定的对象资产边界，同时也便于报告的使用人更加清晰地理解评估对象。资产评估范围是对评估对象组成、结构的进一步补充说明，只在某些情况下两者重合一致，如评估对象为企业可辨识的资产时，其评估对象与评估范围是一致的 - 单项资产：当资产评估对象是单项资产时，评估范围是对该项资产边界的描述。 - 资产组合：当资产组合的评估对象是资产组的权益时， 其评估范围是组成该资产组合的全部资产与负债。 - 整体企业：当评估对象为企业整体权益、股东全部权益和股东部分权益时，评估范围包括被评估企业的全部资产和负债，包括可辨识的资产和不可辨识的资产（如商誉等)。

    范围的确定 - 委托人负债提供 - 评估人关注资产范围与经济行为要求的评估范围一致

    资产评估基准日

    作用： - 明确评估结论所对应的时间：现时性评估、追溯性评估、预测性评估 - 用于确定评估报告结论的使用期限：通常，只有当评估基准日与经济行为实现日相距不超过一年时，才可以使用资产评估报告。

    基准日的选择：委托人的责任，评估人提供专业建议 - 国资评估：大多是现时性评估。 - 国有资产评估备案项目评估基准日应该选择在评估报告使用有效期之前至少4个月； - 国有资产评估核准项目评估基准日至少选择在评估报告使用有效期之前至少5个月。 - 其他资产评估 - 评估基准日的选择与评估报告中引用其他报告基准日的匹配问题

    资产评估报告日

    概念：按照《资产评估执业准则——资产评估报告》的规定，资产评估报告日通常为评估结论形成的日期。

    法律意义：在评估基准日到评估报告日之间，如果被评估资产发生重大变化，评估机构负有了解和披露这些变化以及可能对评估结论产生影响的义务。

    资产评估基准日后的期后事项的处理原则： - 评估机构和评估专业人员需要采用适当的方式，对评估专业人员撤离评估现场后至评估报告日之间， 被评估资产所发生的相关事项以及市场条件发生的变化进行了解，并分析判断该事项和变化的重要性，对于较重大的事项应该在评估报告中进行披露，并提醒报告使用者注意该期后事项对评估结论可能产生的影响； - 如果期后发生的事项非常重大，足以对评估结论产生颠覆性影响，评估机构应当要求评估委托人更改评估基准日重新评估。

    价值类型

    概念：价值类型是指资产评估结果的价值属性及其表现形式。不同价值类型从不同角度反映资产评估价值的属性和特征。不同的价值类型所代表的资产评估价值不仅在性质上是不同的，在数量上往往也存在着较大差异。

    作用：影响评估价值、评估方法、清楚表达结论

    1. 影响和决定资产评估价值的重要因素
    2. 对评估方法有一定的影响
    3. 明确价值类型，可以更清楚地表达评估结论，可以避免评估委托人和其他报告使用人误用评估结论

    价值种类：5种加其他

    1. 市场价值：市场价值是在适当的市场条件下，自愿买方和自愿卖方在各自理性行事且未受任何强迫的情况下，评估对象在评估基准日进行公平交易的价值估计数额。
      1. 市场价值主要受到两个方面因素的影响：其一是交易标的因素，即不同的资产，不同获利能力的资产自然会有不同的市场价值。其二是交易市场的因素，不同的市场可能存在不同的供求关系等因素，对交易标的市场价值产生影响。
      2. 是以公开市场假设作为逻辑基础的。
    2. 投资价值：投资价值与市场价值相比，除受到交易标的因素和交易市场因素影响外，其最为重要的差异是投资价值还受到特定交易者的投资偏好或追求协同因素的影响。
    3. 在用价值：也称使用价值。在用价值是指将评估对象作为企业、资产组组成部分或者要素资产按其正在使用方式和程度及其对所属企业、资产组的贡献的价值估计数额。
    4. 清算价值：清算价值是受外力胁迫、快速变现的退出价。
    5. 残余价值：所谓残余价值实际是将一项资产拆除成零件进行变现的价值。
    6. 其他价值类型：如公允价值

    影响价值类型的因素：目的、市场条件、对象条件、与假设相关性

    1. 评估目的（根本因素）
    2. 市场条件
    3. 评估对象自身条件（自身的功能，使用方式）
    4. 与评估假设的相关性

    具体选择：

    • 市场价值类型的选择：
    • 投资价值类型：通常需要说明选择的理由以及所考虑投资价值包含的与市场价值区别的要素。
    • 在用价值类型：
    • 清算价值：清算价值类型与以清算为目的的评估有联系，但两者并不是直接对应的关系。
    • 残余价值：当评估对象无法或者不宜整体使用，将其拆解分割后的零部件还具有使用价值时，评估专业人员通常应当考虑将评估对象拆零变现，并选择残余价值作为评估结论的价值类型。

    评估假设

    概念：资产评估假设是依据现有知识和有限事实，通过逻辑推理，对资产评估所依托的事实或前提条件作出 的合乎情理的推断或假定。资产评估假设也是资产评估结论成立的前提条件。

    常见评估假设：前提性假设、外部环境假设、具体假设。

    • 评估前提假设：
      • 针对交易市场条件：交易假设、公开市场假设
      • 针对企业等经营主体：持续经营假设、清算假设
      • 针对单项资产：原地使用假设、移地使用假设、最高最佳使用假设、现状利用假设
    • 评估外部环境假设
      • 宏观环境
      • 行业及地区环境
    • 评估具体假设：按照评估目的及评估操作要求针对评估对象的特点所具体使用的评估假设。

    常见评估假设内涵：

    1. 交易假设：最基本的前提假设，假定所有评估标的已经处在交易过程，评估专业人员根据被评估资产的交易条件等模拟市场进行估价。
    2. 公开市场假设：公开市场假设是指资产可以在充分竞争的市场上自由买卖，其价格高低取决于一定市场的供给状况下独立的买卖双方对资产的价值判断。公开市场假设是资产评估假设中其他假设的基本参照。
    3. 持续经营假设：一般不适用于单项资产，通常是收益法评估基础
    4. 有序清算假设：由该经营主体的所有者依法自主组织实施的。
    5. 强制清算假设：清算活动不受破产经营主体的所有者控制。
    6. 原地使用：原地使用是指一项资产在原来的安装地继续被使用，其使用方式和目的可能不变，也可能会改变。
    7. 移地使用：移地续用涉及设备的拆除、迁移和重新安装调试等环节。
    8. 最高最佳：法律允许、技术可能、经济可行
    9. 现状利用：现状利用方式可能不是最高最佳使用方式。

    主要假设的应用：合理、针对、相关

    1. 基本要求：合理性、针对性、相关性
    2. 需要考虑的基本因素：目的、对象
    3. 交易假设：最基本的假设。适用于交易和产权变动。
    4. 公开市场假设：使用频率较高
    5. 持续经营假设：针对经营主体，不是针对单项资产。如果需要考虑在收购交易完成后，实现了有关的协同效应后的权益价值，就不能采用“现状持续经营”假设，而应当以并购完成后的状态持续经营为假设条件，以便评估其投资价值。

    市场法

    概念：

    • 替代原则是支持原则
    • 市场法在不同的应用领域存在不同的表现形式和称谓。例如，企业价值评估中的交易案例比较法和上市公司比较法，机器设备评估中的直接比较法、相似比较法、比率估价法，矿业权评估中的可比销售法、单位面积探矿权价值评判法、资源品级探矿权价值估算法等。
    • 较容易被各方接受

    应用前提：

    1. 公开市场、活跃交易
    2. 有关交易的必要信息可以获得：有可比性、可合法获得、充分

    基本步骤

    1. 选择参照物：可比性、尽可能多
    2. 选择比较因素：选择影响较大的因素，多方面对比
    3. 指标对比和量化差异：将差异数量化货币化
    4. 分析已经量化指标之间的差异
    5. 综合分析确定评估结果：如果参照物与评估对象可比性都很好，评估过程中没有明显的遗漏或疏忽，评估专业人员可以用算术平均法或加权平均法等方法将初步结果转换成最终评估结果。

    可比因素

    1. 单项资产评估
      1. 资产的功能
      2. 实体特征和质量
      3. 市场条件：市场供不应求时，价格偏高；供过于求时，价格偏低
      4. 交易条件：主要包括交易批量、交易动机、交易时间等。
      5. 具体可比因素：房地产评估中的区位因素，机器设备评估中的制造厂家、资产规格型号等
    2. 企业价值评估
      1. 行业标准
      2. 财务标准
      3. 其他重要交易条款等：应考虑股权评估交易的背景、交易日期、交易价格、收购股权的比重、影响交易价格的其他交易条款

    直接比较法具体方法：

    1. 现行市价法：评估对象或参照物在评估基准日的现行市场价格需要与评估对象的价值内涵相同
    2. 市价折扣法：仅存在交易条件差异
    3. 功能价值类比法：1. 线性关系、2. 指数关系
    4. 价格指数法：1. 定基、2. 环比
    5. 成新率价格调整法：适用仅成新率有差异
    6. 市场售价类比法：在功能、市场条件和销售时间等方面的差异，通过对比分析和量化差异，调整估算出评估对象价值的一种方法。
    7. 价值比率法：与某一经济参数或经济指标的比率作为乘数或倍数
      1. 按价值对应口径分：全投资比率、股权投资比率
      2. 按指标参数的性质分：盈利价值比率、收入、资产
      3. 应用：上市公司比较法、交易案例比较法
      4. 关键点：
        1. 数据来源及可靠性
        2. 可比标准确定
        3. 参照物的质量重于数量
        4. 价值比率的选取

    间接比较法：不常用

    • 是利用资产的国家标准、行业标准或市场标准（标准可以是综合标准，也可以是分项标准）作为基准， 分别将评估对象和参照物整体或分项对比打分从而得到评估对象和参照物各自的分值。再利用参照物的市场交易价格以及评估对象的分值与参照物的分值的值（系数）求得评估对象价值的一种评估方法。
    • 该方法并不要求参照物与评估对象必须一样或者基本一样。只要参照物与评估对象在大的方面基本相同或相似，通过评估对象和参照物与国家、行业或市场标准的对比分析，掌握参照物与评估对象之间的差异，在参照物成交价格的基础上调整估算评估对象的价值。

    适用范围

    收益法

    概念

    • $年金现值系数=A\frac{1-(1+r)^{-n}}{r}$
    • 年金终值系数=$A\frac{(1+r)^n-1}{r}$
    • 1＋名义利率＝（1＋实际利率）×（1＋通货膨胀率）

    1. 收益可预期并计量
    2. 所对应的风险能够度量
    3. 期限可以确定或合理预期

    1. 搜集或验证与评估对象未来预期收益有关的数据资料；
    2. 分析测算评估对象的未来预期收益；
    3. 分析测算评估对象预期收益持续的时间；
    4. 分析测算折现率或资本化率；
    5. 用折现率或资本化率将评估对象的未来预期收益折算成现值；
    6. 分析确定评估结果。

      1. 特点：未来预期，不是历史和现在。资产的客观收益，
      2. 预测方法
        1. 时间序列法：适用条件：资产的收益随着时间的推移，呈现出平稳增长趋势
        2. 因素分析法：间接预测法，优势：但适用面比较广，预测结果也具有一定的客观性，被广泛采用。
    2. 折现率
      1. 本质：折现率是一种投资回报率
      2. 构成：无风险报酬率，风险报酬率

    适用范围：

    • 单项资产评估：①无形资产；②具有收益性的房地产； 适用于轻资产类型的企业价值评估；③对于可出租的机器设备或可独立产生现金流的生产线、成套设备（单台机器设备很难独立产生收益）；④其他资产，如非上市交易的股票、 债券、长期应收款、长期股权投资等。 轻资产类型的企业价值评估。
    • 优势：较适宜于那些形成资产的成本费用与其获利能力不对称，成本费用无法或难以准确计算，存在无形资源性资产以及具有收益能力的资产，例如企业价值、无形资产、资源性资产等的价值评估。 收益法从本质上体现了企业作为经营主体的存在目的，较为真实和准确地体现了企业的资本化价值， 能够为所有者或潜在投资者提供较为合理的预期，有助于投资决策的正确性，因而，容易被买卖双方接受。
    • 局限：
      1. 要有收益
      2. 有一定主观性
      3. 需要一定市场条件，否则无数据

    成本法

    概念

    成本法是指通过估测评估对象的重建或重置成本，然后估测评估对象已存在的各种贬值，并将其从重置成本中予以扣除而得到评估对象价值的各种评估技术方法的总称。

    基本思路

    评估价值＝重置成本－实体性贬值－功能性贬值－经济性贬值

    应用前提

    1. 能使用或在用、2. 能通过重置获得、3. 重置成本及贬值能合理估算

    基本步骤

    1.确定评估对象，并估算重置成本； 2.确定评估对象的使用年限； 3.测算评估对象的各项损耗或贬值额； 4.测算评估对象的价值。

    重置法参数

    1. 重置成本：重置成本应当是社会一般生产力水平的客观必要成本，而不是个别成本。
      1. 复原重置成本：是指采用与评估对象相同的材料、建筑或制造标准、设计、规格及技术等，以现时价格水平重新购建与评估对象相同的全新资产所发生的费用。 用于评估对象的效用只能通过按原条件重新复制评估对象的方式提供。
      2. 更新重置成本：是指采用与评估对象并不完全相同的材料、建筑或制造标准、设计、规格和技术 等，以现行价格水平购建与评估对象具有同等功能的全新资产所需的费用。 适用于用当前条件所重置的资产可以提供与评估对象相似或者相同的功能，并且更新重置成本通常低于其复原重置成本。
    2. 实体性贬值：资产的实体性贬值，也称有形损耗，是指资产由于使用及自然力的作用导致的资产的物理性能的损耗或下降而引起的资产价值损失。
    3. 功能性贬值
      1. 包含内容：1. 超额投资成本、2. 超额运营成本
      2. 体现在资产的市场价值中：如果重置成本中已经包含了功能性贬值因素，则在评估中就不需要另行扣除功能性贬值；
    4. 经济性贬值：资产的经济性贬值指由于外部条件的变化起资产收益、资产利用率发生持续减少、下降或者闲置等而 造成的资产价值损失。
      1. 包括：①市场竞争性因素引起的经济性贬值。 ②政策因素引起的经济性贬值。
      2. 表现形式：①资产利用率下降，甚至闲置。 ②资产的运营收益减少。

    具体方法：重置核算、价格指数、功能价值类比、统计分析法

    1. 重置核算法：考虑合理收益。重置核算法既考虑了价格因素，也考虑了生产技术进步和劳动生产率的变化因素，因而可以用来估算更新重置成本
    2. 价格指数法：
      1. 适用前提：①既无法获得处于全新状态的评估对象的现行市价； ②也无法获得与评估对象相类似的参照物的现行市价； ③可以利用与资产有关的价格变动指数计算评估对象的重置价值。
      2. 价格指数法估算的重置成本仅考虑了价格变动因素，因而确定的是复原重置成本
    3. 功能价值类比法：
      1. 生产能力比例法
      2. 规模经济效益指数法：我国到目前为止尚未有统一的经验数据，评估过程要谨慎使用这种方法。
    4. 统计分析法

    贬值估算

    1. 实体性贬值：1. 观察法、2. 使用年限法、3. 修复费用法
    2. 功能性贬值：超额投资成本，超额运营成本（税后，折现）
    3. 经济性贬值：
      1. 经济贬值率：$[1-(预计可利用产能/原设计产能)^x]\times100\%$
      2. 收益额减少（税后，折现）
    4. 综合成新率估算：在资产评估实践中，综合成新率可以通过将年限成新率和勘查成新率加权平均得到。

    适用范围：可替代、重置无障碍、物化易计量

    • 用于资产的功能作用具有可替代性、资产重置没有法律和技术障碍、重置资产所需的物化劳动易于计量的评估对象。
    • 通常可以被用在评估某些没有获利能力的无形资产，或者正处于使用初期的这类资产，比如新注册但还未运用于任何产品的注册商标
    • 优势：成本法是资产评估中最为基础的评估方法。
      1. 它充分考虑了资产的损耗，使得评估结果能反映市场对于获得某单项资产愿意付出的平均价格，有利于评估单项资产和具有特定用途的资产；
      2. 在无法预测资产未来收益和市场交易活动不频繁的情形下，给出了比较客观可行的测算思路和方法。
      3. 成本法评估企业价值为可能的破产清算、资产分割提供了一定的价值参考。 局限
      4. 成本法的理论基础是成本价值论，因而测算出企业价值无法从未来收益的角度反映企业真实能为其投资者或所有者带来的收益。
      5. 轻资产企业成本法评估值与收益法或市场法得出的结果可能差异极大。

    选择

    考虑因素 1. 评估目的和价值类型； 2. 评估对象； 3. 评估方法的适用条件； 4. 评估方法应用所依据数据的质量和数量等； 5. 其他因素。

    可以采用一种评估方法的情形 ①基于相关法律、行政法规和财政部部门规章的规定； ②由于评估对象仅满足一种评估方法的适用条件； ③因操作条件限制而采用一种评估方法。

    第7章：资产评估报告与档案15

    资产评估报告是指资产评估机构及其资产评估专业人员遵守法律、行政法规和资产评估准则，根据委托履行必要的评估程序后，由资产评估机构对评估对象在评估基准日特定目的下的价值出具的专业报告。 评估报告中需满足报告使用人的合理要求，提供必要信息，使评估报告使用人正确理解和使用评估结论。 评估报告应当按照一定的格式和内容编写。

    分类：

    按法律地位： - 法定业务评估报告：涉及国有资产的评估业务并不一定都属于法定业务，如以财务报告为目的的资产评估就不属于法定 评估业务。《公司法》规定，对作为出资的非货币财产应当评估作价。该类资产评估业务也属于法定评估业务。 - 非法定业务评估报告： 按评估对象：整体、单项 按评估基准日：追溯性、现时性、预测性 按监管主体要求：国有资产业务、非国有资产

    评估报告的基本要求

    1. 陈述的内容应当清晰、准确，不得有误导性的表述
    2. 应当提供必要信息，使资产评估报告使用人能够正确理解和使用评估结论
    3. 详略程度可以根据评估对象的复杂程度、委托人的要求合理确定
    4. 评估程序受限对评估报告出具的要求：①经采取措施弥补程序缺失，且未对评估结论产生重大影响的，可以出具资产评估报告，但应当在资产评估报告中说明资产评估程序受限情况、处理方式及其对评估结论的影响。 ②如果程序受限对评估结论产生重大影响或无法判断其影响程度，不应出具资产评估报告。
    5. 签字印章要求：①在资产评估机构履行内部审核程序后 ②至少两名承办该项业务的资产评估专业人员签名＋加盖资产评估机构印章 ③法定评估业务至少两名承办该项业务的资产评估师签名＋加盖资产评估机构印章 ④资产评估专业人员只能对本人参与过的评估项目签署评估报告，不得代人签署，亦不得允许他人以本人名义签字。
    6. 语言及汇率要求
    7. 评估结论的使用有效期：现时性评估报告评估结论有效的前提： ①评估基准日至经济行为日期间市场无重大变化； ②评估基准日至经济行为日间隔不超过一年。

    评估报告的基本内容

    1. 封面
    2. 标题：企业名称＋经济行为关键词＋评估对象＋资产评估报告
    3. 文号：资产评估机构特征字、种类特征字、年份、报告序号
    4. 声明：
      1. 编制依据、
      2. 按规定使用、
      3. 界定报告使用人、
      4. 理解评估结论、
      5. 报告的关注点、
      6. 原则及法律责任、
      7. 其他需要声明内容。
    5. 摘要：合理确定摘要中需要披露的其他信息。通常要披露的内容有:①评估目的;②评估对象和评估范围;③价值类型及其定义;④评估基准日;⑤评估方法;⑥评估结论。
    6. 正文
      1. 委托人及其他资产评估报告使用人
      2. 评估目的：报告载明的评估目的应当唯一，有利于评估结论有效服务于评估目的。
      3. 评估对象和评估范围
      4. 价值类型
      5. 评估基准日：应当与资产评估委托合同约定的评估基准日保持一致。
      6. 评估依据：应当说明资产评估采用的法律依据、准则依据、权属依据及取价依据等。
      7. 评估方法
        1. 与评估目的、评估时的市场条件、被评估对象的具体状况，以及由此所决定的资产评估价值类型的适应性。
        2. 各种评估方法运用所需的数据资料及主要经济技术参数的收集条件对选择评估方法的制约。
        3. 评估方法运用的条件和程序要求对选择评估方法的约束。
        4. 市场法：参照物（交易案例）的选择原则、比较分析、调整因素等
        5. 收益法：技术思路、主要测算方法、模型或计算公式、明确预测收益的类型、预测方法与过程、 折现率的选择和确定等
        6. 成本法：估算公式、重置价值确定方法、成新率的确定方法
        7. 多种方法：确保满足各方法使用条件要求和程序要求，对各种评估方法得到的评估结果进行比较， 分析可能存在的问题并做相应的调整，确定最终评估结论
      8. 评估程序实施过程和情况
      9. 评估假设：评估假设分为一般性假设和针对性假设。
      10. 评估结论：
        1. 形式：1.文字和数字形式，明确结论使用有效期。2.确定的数值、区间值、其他形式。
        2. 确定的数值可以采用以下两种方式得到： ①采用两种以上评估方法的，选取其中相对合理方法的结果作为评估结论。 ②对不同方法的评估结果，采用算术平均、加权平均、求取中位数等方法综合得出评估结论。
      11. 特别事项说明：已发现可能影响评估结论，但非执业水平和能力所能评定估算的有关事项。
        1. 权属等主要资料不完整或存在瑕疵
        2. 委托人未提供的其他关键资料情况
        3. 未决事项、法律纠纷等不确定因素
        4. 重要的利用专家工作及相关报告情况
        5. 重大期后事项
        6. 评估程序受限有关情况、评估机构采取的弥补措施及对评估结论影响的情况。
        7. 其他需要说明的事项。
      12. 资产评估报告使用限制说明
      13. 资产评估报告日：通常为评估结论形成的日期，可以不同于资产评估报告的签署日。
      14. 资产评估专业人员签名和资产评估机构印章
    7. 评估报告附件
      1. 主要权属证明资料
      2. 承诺函
      3. 资产评估机构及签名资产评估专业人员的备案文件或者资格证明文件：①评估机构从事资产评估业务的备案公告复印件 ②从事证券服务业务的机构备案名单复印件（开展相关资产评估业务时适用） ③资产评估师的职业资格证书登记卡复印件
      4. 资产评估汇总表或明细表
      5. 资产账面价值与评估结论存在较大差异的说明
      6. 实务中，通常还会将委托人和被评估单位或产权持有人的法人营业执照、评估机构的营业执照复印件也作为评估报告的附件。

    国资评估报告特殊要求

    1. 构成
      1. 企业国有资产评估报告
      2. 金融企业国有资产评估报告
      3. 文化企业国有资产评估报告
      4. 行政事业单位国有资产评估报告等
    2. 特殊要求
      1. 评估报告声明
        1. 编制依据
        2. 使用理解
        3. 评估责任
        4. 评估资料：评估对象涉及的资产、负债清单由委托人、被评估单位申报并经其采用签名、盖章或法律允许的其他 方式确认；
        5. 利益关系：资产评估机构及资产评估师与资产评估报告中的评估对象没有现存或者预期的利益关系；与相关当事人没有现存或者预期的利益关系，对相关当事人不存在偏见。
        6. 必要程序：对资产评估报告中的评估对象及其所涉及资产进行现场调查；已经对评估对象及其所涉及资产的法律权属状况给予必要的关注，对评估对象及其所涉及资产的法律权属资料进行了查验，对已经发现的问题进行了如实披露，并且已提请委托人及其他相关当事人完善产权以满足出具资产评估报告的要求。
        7. 提示
      2. 摘要
      3. 正文：
        1. 绪言
        2. 委托人、被评估单位和资产评估委托合同约定的其他资产评估报告使用人的概况
        3. 评估对象和范围
        4. 评估基准日：①本项目评估基准日是 XXXX 年 XX 月 XX 日。 ②确定评估基准日所考虑的主要因素（如经济行为的实现、会计期末、利率和汇率变化等）。
        5. 评估依据：经济行为依据（包括两个方面：一是有效批复文件，二是其他文件资料。）、法律法规依据、评估准则依据、权属依据、取价依据
        6. 评估结论：
          1. 表达方式种类
            1. 确定值：作为企业国有资本出资额或者股份数额的依据（评估确认价格）
            2. 区间值：《企业国有资产评估报告指南》规定，境外企业国有资产评估报告的评估结论可以用区间值表达。
          2. 披露内容的要求：
            1. 采用资产基础法进行企业价值评估，应当以文字形式说明资产、负债、所有者权益（净资产）的账面价值、评估价值及其增减幅度，并同时采用评估结论汇总表反映评估结论。
            2. 单项资产或者资产组合评估，应当以文字形式说明账面价值、评估价值及其增减幅度。
            3. 采用两种以上方法进行企业价值评估，除单独说明评估价值和增减变动幅度外，还应当说明两种以上评估方法结果的差异及其原因和最终确定评估结论的理由。
            4. 存在多家被评估单位的项目，应当分别说明评估价值。
            5. 评估结论为区间值的，应当在区间之内确定一个最大可能值，并说明确定依据。
        7. 特别事项说明：国有资产评估业务，要求资产评估报告应当说明评估程序受到的限制、评估特殊处理、评估结论瑕疵等特别事项，以及期后事项。
        8. 资产评估师签名和资产评估机构印章
      4. 附件要求：审计报告不仅是企业申请备案核准的必报资料之一，而且是评估工作的基础依据。附件通常包括下列内容：①与评估目的相对应的经济行为文件；②被评估单位专项审计报告；③委托人和被评估单位产权登记证；④签名资产评估师的承诺函；⑤资产评估机构法人营业执照副本；⑥资产评估委托合同；⑦引用其他机构评估报告的批准（备案）文件；⑧其他重要文件。
    3. 评估明细表要求

    第1章：流动负债及负债评估10

    概述

    流动资产内容：库存现金、各种银行存款以及其他货币资金、交易性金融资产、应收票据、应收账款、预付账款、其他应收款、待摊费用、预付费用、存货以及其他流动资产等。

    流动资产特点：1.循环周转速度快。2.变现能力强。3.占用形态同时并存又相继转化。4.波动性

    流动资产评估对象：一般作为单独的评估对象，不需要以其综合获利能力进行综合性价值评估。

    流动资产评估特点： 1. 评估基准日要尽可能与评估结论使用时点接近。 2. 量大类繁，一般需进行分类清查和评估。 3. 评估实务受企业运营的牵制较大，对企业流动资产会计核算资料的依赖度高。 4. 在正常情况下，流动资产账面价值基本可以反映现值。

    流动资产的评估目的和方式 - 持续经营 - 在用用途：重置成本 - 转移使用或出售：按变现净值 - 清算：快速变现，按快速变现净值

    企业负债的定义： 流动负债 长期负债：长期借款、应付债券、长期应付款及递延收益。

    企业负债特点： 1. 负债是企业承担的现时义务 2. 负债的清偿预期会导致经济利益流出企业 3. 负债是由过去的交易或事项形成的

    企业负债评估特点： 理论上：反映市场价值 实践中：反映现时义务。不适用于三大方法

    存货评估

    清查核实方法 1. 界定评估对象，确定评估范围 2. 确定基准日 1. 接近会计报表，方便利用 2. 最好选择在评估工作期间或接近 3. 尽可能当地，减少价格调整工作量 3. 清查核实：有没有，属于谁 4. 选择评估方法：一是根据评估目的，二是根据不同种类存货的特点。 1. 价格变化大：市场法 2. 不大：成本法

    库存材料： 1. 近期购进：成本法市场法均可 2. 批次间隔长价格变化大：市场价格，存货计价方式不影响评估结果 3. 缺乏准确现价：替代品修正，同类商品物价指数 4. 呆滞材料：扣除贬值

    低耗品 1. 在库的：同库存材料 2. 在用的：可用成本法考虑成新率 3. 一般不考虑功能和经济性损耗，成新率应该按照实际损耗程度确定，不能简单按照摊销方式确定。

    在产品 1. 成本法： 1. 生产周期长的：按评估时价格重置 2. 生产周期短的：可以根据账面值调整 1. 根据价格变动系数调整，适用于生产经营正常，会计核算水平较高的情形。 2. 按社会平均消耗定额和现行市价计算评估值。 3. 按在产品的完工程度计算评估值。 2. 市场法：按同类在产品和半成品的市价，扣除销售过程中预计发生的相关费用后计算评估值。

    产成品及库存商品 1. 成本法 1. 基准日与完工日接近：产品账面单位成本 2. 间隔较长：料工费现价 2. 市场法： 1. 使用价值；分析市场供求关系和被评估产品的前景；所选择的价格应是公开市场上形成的近期交易价格； 2. 残缺情况通过调整系数予以调整。 3. 考虑因素：①库存商品的使用价值。根据对产成品本身的技术水平和内在质量的技术鉴定，确定产成品是否具有使用价值以及产品的实际等级，以便选择合理的市场价格。②分析市场供求关系和被评估产品的前景。③所选择的价格应是在公开市场上所形成的近期交易价格，非正常交易价格不能作为评估的依据。④对于产品技术水平先进，但产成品外表存有不同程度的残缺的情况，可根据其损坏程度，通过调整系数予以调整。

    房地产开发企业存货评估方法 1. 原材料类、设备类、开发用品类： 1. 如外购原材料、辅助材料、燃料、在库低耗。 2. 可用一般存货评估方法 2. 在产品类 1. 没有开工土地：市场法或假设开发法 2. 没有完工的土地和房屋：假设开发法，成本法 3. 产成品类： 1. 出售：如商品房，市场法 2. 出租：收益法

    货币资金：不存在价值估算，仅需核对数额。

    应收账款：应收账款的评估，不是对债权金额进行重新认定，评估的是“风险损失”以及“未来现金资产（未来 回收价值的现值）”。 - ＝应收款项账面余额-已确定的坏账损失-预计可能发生的坏账损失 - 环账比例法、账龄分析法 - 应收账款评估值不必考虑坏账准备数额。 - 欠款方承认债务的存在、认可债务账面记录，并不说明债权方能够全额收回欠款，有偿还的动机和足够的偿还能力才是全额收回欠款的真正依据。

    应收票据： 1. 按票据的本利和计算 2. 按应收票据的贴现值计算：减去贴现息

    待摊费用和预付费用：如果已经消耗了，价值为0

    负债评估

    基本思路：判断是否为企业实际承担的负债以及在评估基准日负债数额是否正确。

    应付及预收款：应特别关注是否存在不需要支付的债务，如有，评估为零。

    应付职工薪酬：一般以核实后的余额作为评估值。

    银行借款：考虑利息及折现等 - 短期借款：利息是否已经预提 - 长期借款：应按评估基准日的适用折现率进行折现，并将折现后的现值作为长期借款的评估价值。 实务操作中，通常简化为将核实后的账面余额作为评估值。

    应付票据： - 应付商业承兑汇票到期，如企业无力支付票款，应将应付票据按票面金额转作应付账款； - 应付银行承兑汇票到期，如企业无力支付票款，应将应付票据的票面金额转作短期借款。

    应付股利：确认账面余额

    应交税费：企业应交的税费中，除印花税、耕地占用税以及其他不需要预计应交数的税费外，其他税费均需要通 过“应交税费”科目核算。

    长期应付款： 1. 项目未完工：按账面值保留评估值 2. 已完成验收：评估值按照资产剩余寿命期所对应的价值进行确认

    递延收益：核算企业确认的应在以后期间计入当期损益的政府补助。 1. 与资产相关 1. 项目未完工：按账面值保留评估值 2. 已完成验收：评估值按照资产剩余寿命期所对应的价值进行确认 2. 与收益相关： 1. 用于补偿已发生的费用损失：评估值仅保留所得税 2. 以后期间的：评估值按照相关费用期间所对应的价值进行确认。 3. 政府补助如果确认为不征税收入，可以不考虑递延所得税负债，即上述用于补偿企业已发生 的相关费用或损失的递延收益的评估值应为零

    第2章：长期投资性资产评估5

    债券投资评估

    评估基本思路：如果债券可以在市场上自由买卖、变现时，其就是债券的评估值。否则，就需要通过其他 方法评估。 - 上市交易：市场法 - 非上市：收益法。无风险报酬率通常以银行存款利率、国债利率或政府发放短期债券的利率等为准。

    股票投资评估

    特点：高风险高收益。股票价值的评估和内在价值、清算价格和市场价格有着较为密切的联系。

    评估思路 - 正常情况：现行市价法（基准日收盘价） - 非正常情况：内在价值法 - 控股为目的：收益法

    非上市交易股票的价值评估：一般收益法 - 普通股：固定股利，股利增长（股利增长率的计算方法：统计分析法，趋势分析法），分段型 - 优先股：评估优先股主要是判断股票发行主体是否有足够税后利润用于优先股的股息分配。

    投资基金

    分类：公募vs私募。证券投基金vs另类投资基金。开放vs封闭。契约型vs公司型vs有限合伙型。

    证券投资基金特点： - 投资基金评估是对基金资产净值（份额）的评估，是指通过对基金所拥有的全部资产及全部负债按一定的原则和方法进行估算，进而确定基金资产价值的过程。 - 投资基金净值不仅仅为基金资产的价值减去基金负债的价值，还应扣除对应的管理费和管理人对应的超额收益（CARRY）。 - 建立在底层资产估值基础上的，要尽可能穿透。

    证券投资基金应注意的问题： 1. 关注协议章程、了解投资人、委托人、管理人、托管人在利益分配、管理费支付 等方面的权利和义务。 2. 应根据投资基金底层资产的资产状况、经营状态，对底层资产根据其不同类型采用不同的评估方法。 3. 对非公开投资基金尽可能取得其财务报告，其财务报告中一般都会对资产进行估值，评估人员应关注其财务报告中资产估值的合理性。

    证券投资基金评估：基金资产净值＝基金资产－基金负债。基金份额净值＝基金资产净值/基金总份额

    合伙制基金评估：基金资产净值＝项目价值总和 ＋ 其他资产价值 － 基金费用等负债。对于合伙制基金，普通合伙人 GP 和有限合伙人 LP 对应的份额估值有不同的价值内涵，即不必然等于股东全部权益与股权比例的乘积。

    长期股权投资的评估

    基本特点： 1. 对资本或权益的评估。 2. 可能是对被投资企业获利能力的评估，也可能是为间接提高投资方自身盈利能力的战略投资价值的评估。 3. 通常是建立在被投资企业持续经营的基础上。 4. 通常是基于被投资企业权益的可分性，即企业的权益可分为股东全部权益和股东部分权益，不同的权益份额可能有不同的价值内涵。

    关注的问题： 1. 应当把被投资单位作为一个有机整体。 2. 应关注公司章程或投资协议。 3. 应关注公司最新的工商登记情况和近期的董事会决议等材料，了解被评估单位股权结构及股东权益。 4. 应关注评估范围内的重要资产和负债。 5. 应关注可能影响评估结论的重要事项。 6. 应关注关联交易的影响。 7. 应关注亏损企业采用收益法评估的适用性。

    具有控制权的评估 - 特点：程序不受限，关注权属资料完善情况，可能存在控股权溢价。 - 评估程序： - 明确范围：通常为被投资单位法人全部资产，包括企业拥有权属清晰并投入经营或未投入经营的资产，企业实际拥有但尚未办理产权登记的资产 - 核实投资协议，账面记录 - 核对持股比例 - 选择评估方法，编制评估计划：分析收益法、市场法和资产基础法方法的适用性，综合判断，恰当选择评估方法。 - 确定价值 - 清查核实方法：核对、抽查、尽职调查、访谈 - 评估结论确定： - 资产基础法VS收益法：应考虑两者的评估对象与评估范围的一致性。 - 市场法VS收益法：关注对非经营性、溢余资产负债的判断是否一致。 - 市场法VS资产基础法：市场法的价值比率选取了市净率时，与资产基础法如何衔接的问题就更加突出。

    缺乏控制权的评估 - 特点：评估程序及方法选择可能受限。应关注流动性的影响。 - 评估程序： - 确定评估范围： - 单独评估：评估范围通常为被投资单位产权涉及的全部资产和负债，包括企业拥有权属清晰并投入经营或未投入经营的资产，企业实际拥有但尚未办理产权登记的资产。 - 非单独评估：评估范围可以是被投资单位的全部资产和负债，也可以是长期股权投资本身。 - 核实投资协议，账面记录 - 单独评估：核实内容与具有控制权的股权一致。 - 非单独：应向母公司取得股权投资的初始协议。 - 核对持股比例 - 选择评估方法，编制评估计划 - 单独评估：三种方法的适用性 - 非单独评估：判断市场法及收益法的适用性 - 确定评估值 - 清查核实方法：核对、抽查、尽调、访谈 - 核对： - 单独评估：主要针对被投资企业 - 非单独：主要针对母公司核算及管控长期股权投资的全部档案资料及被投资单位的近期财务报告，必要时向被投资企业发函确认。 - 抽查： - 单独评估：同有控制权股权评估 - 非单独：抽查主要针对母公司核算长期股权投资的各期会计凭证，以及母子公司关联往来科目的会计凭证，核实其历史成本或核算入账的背景以及关联往来情况。 - 尽调： - 非单独：独立调查为主，调查问卷为辅。 - 访谈： - 非单独：访谈对象为母公司负责该长期股权投资核算的财务人员及母公司派驻被投资单位任职高管。 - 评估方法： - 资产基础法：不为首选，对于停业或拟清算的仅能用此。关注合理确认负债。 - 收益法：适用于缺乏控制权的部分权益评估。 - 市场法：需要找到可比交易案例或上市公司。 - 其他方法：可使用被投资企业评估基准日财务报表分析确定长期股权投资评估价值，并说明不能单独评估的理由和使用财务报表分析确定股权价值的过程。

    第3章：机器设备评估20

    概述

    机器设备评估定义：单独的机器设备或者作为企业资产组成部分的机器设备价值进行评估

    评估特点： 1. 一般不具备独立的获利能力，通常用重置成本法和市场法进行评估，收益法受到一定的限制； 2. 存在形式多样 3. 分为动产和不动产 4. 机器设备除了实体性贬值外，还有功能性贬值和经济性贬值，很多因素都对设备价值产生影响。

    评估对象：区分评估对象是单台机器设备(独立发挥作用)以及机器设备组合(多设备组成的有机整体) 1. 对于己安装的在用设备，是否包含设备基础、附属设施等。 2. 是否包括操作软件、技术数据和专利等无形资产。

    价值类型： 1. 做为整体资产组成部分的：依据替代原则，评估对象是其续用价值 2. 脱离整体而独立存在的：变现价值，在二手市场能实现的价值 3. 交易受限的： 1. 快速清算价值 2. 有序清算价值：允许一个适当的时限 3. 原地复用清算价值：安装成本很高的

    评估假设 1. 持续使用假设与变现假设 2. 现行用途与改变用途假设 3. 原地使用与移地使用假设

    机器设备评估的清查核实：存在、状态、权属

    目的：1.确定机器设备是否存在。 2.明确机器设备的技术及使用状态。3.查验机械设备的法律权属资料。

    基本要求： - 现场逐项调査或者抽样调查，确定机器设备是否存在、 明确机器设备存在状态并关注其权属。 - 应当充分考虑抽样风险。因客观原因限制，无法实施现场调查的，应当采取措施加以判断，并予以披露。

    清查核实内容 1. 实物状况 2. 使用状况 3. 管理状况： 1. 机器设备的维护：分为日常维护和定期维护两种。 2. 检查：分为日常检查、定期检查、精度检查和法定检查等 3. 修理：预防性修理、事后修理、 改善修理和质量修理。 4. 更新改造 5. 报废。 4. 会计核算 5. 产权权属资料

    方法：一般可以采用逐项调查或抽样调查的方法进行，确定机器设备是否存在，具体状况和权属情况。

    手段：通常采用现场观察，也可以采用远程视频手段实时观察；也可以借鉴档案资料，利用检测或鉴定意见等。

    1. 重置成本法

    基本概念：通过估算机器设备的重置成本，然后扣减其在使用过程中自然磨损、技术进步或外部经济环境导致的各种贬值，即设备的实体性贬值、功能性贬值、经济性贬值

    重置成本：复原重置成本、更新

    1. 复原重置成本：制造工艺、材料与原来完全相同，只需要调整人、料、机的价格水平即可。
    2. 更新重置成本：由于技术进步，无法（或没有必要）确认复原重置成本

    重置成本估算：直接、间接、资金成本

    1. 直接成本
      1. 本体重置成本
      2. 运杂费
      3. 安装费
      4. 基础费
      5. 其他合理成本
    2. 间接成本
      1. 管理费
      2. 设计费
      3. 工程监理费
      4. 保险费
    3. 资金成本：资金占用费

    直接成本：本体、运杂、安装、基础、其他

    1. 设备本体重置成本：方法有市场途径询价法、物价指数调整法、重置核算法、综合估价法、重量估价法、类比法—指数估价法等六种。
      1. 市场途径询价法：
        1. 市场询价：
          1. 公开市场：直接从市场获得，同等条件下选最低
          2. 专用设备：向厂家询价不宜直接用，向最近购买的客户了解实际成交价
        2. 使用价格资料：目录、广告等
      2. 物价指数调整法：定基、环比。选取的是某一类设备的综合物价指数，不是某个设备的物价指数。
        1. 选取的物价指数应与评估对象配比；
        2. 审查历史成本的真实性；
        3. 物价指数法不适用于确定更新重置成本，也不能作为衡量复原重置成本和更新重置成本差异的手段；
        4. 购买时间较长，采用综合物价指数或对高通货膨胀期设备要相当谨慎，并尽可能和别的方法校核；
        5. 进口设备应使用进口国的分类物价指数。
      3. 重置核算法：通过分别测算机器设备的各项成本费用来确定设备本体重置成本的方法。 一般用于确定非标准的、自制的设备。
      4. 综合估价法：通过确定设备的主材费用和主要外购件费用，计算出设备的完全制造成本，并考虑企业利润和设计费等费用，确定设备本体的重置成本。
      5. 重量估价法：假设人工费、车间经费、企业管理费及设计费是设备材料费的线性函数，根据相似设备的统计资料计 算出单位重量的综合费率，以设备的重量乘以综合费率，并考虑利润和税金，并根据设备的复杂系数进行适当调整后，确定设备本体的重置成本。简单快速，适用于材料单一、制造简单、技术含量低的设备本体重置成本的计算。
      6. 类比法——指数估价法：根据相似设备的价格来确定重置成本的方法。
    2. 运杂费：生产厂家到安装使用地点所发生的装卸、运输、采购、保管、保险及其他有关费用。
    3. 安装费：
    4. 基础费:
    5. 进口设备从属费用：进口设备的从属费用包括国外运费、国外运输保险费、关税、消费税、增值税、银行财务费、公司代 理手续费，对车辆还包括车辆购置税等。
      1. 离岸价（FOB）=货价；
      2. 到岸价（CIF）=货价+国外运输费+国外运输保险费。
      3. 国外运费=海运费＝设备的离岸价×海运费率
      4. 国外保险费＝（设备的离岸价＋海运费）×保险费率

      6. 消费税＝（关税完税价＋关税）/（1- 消费税税率）×消费税税率
      7. 增值税＝（关税完税价＋关税＋消费税）×增值税税率

      9. 外贸手续费 ＝设备到岸价×外贸手续费率
      10. 车辆购置税＝（到岸价人民币数＋关税＋消费税）×车辆购置税税率

    设备的磨损与补偿

    1. 磨损
      1. 有型磨损：实物形态磨损
        1. 在使用过程中，设备的零件由于摩擦、振动、腐蚀和疲劳等产生磨损。
        2. 设备在闲置过程中，由于自然力的作用而腐蚀，或由于管理不善和缺乏必要的维护而自然丧失精度和工作能力，使设备遭受有形磨损。
      2. 无形磨损：价格下降或技术进步
        1. 由于相同结构设备重置价值的降低而带来的原有设备价值的贬值， 一般不需要更新。
        2. 由于出现了生产率更高、经济性能更好的设备，原设备经济效果降低。又称为功能性磨损
    2. 补偿
      1. 有形磨损：修理、更换，更新
      2. 无形磨损：
        1. 第一种无形磨损不存在磨损补偿的问题
        2. 第二种无形磨损可采用更新或技术改造。

    实体性贬值：观察法、使用寿命法、修复费用法

    有形磨损使得设备的生产能力下降或使用价值降低，由此引起的贬值称为实体性贬值（Dp），或物理性贬值。 1. 观察法：定性的德尔菲法，量化的模糊综合判断法 2. 使用寿命法 3. 修复费用法：可修复性损耗、不可修复性

    功能性贬值：超额投资、超额运营

    1. 第一种：反映为超额投资成本，用复原重置成本与更新重置成本之差表示超额投资成本。对于大部分通用设备，重置成本一般根据现行市场价格确定，这个价格中已经反映了第Ⅰ种功能性贬值。
    2. 第二种：超额运营成本
      1. 分析比较机器设备的超额运营成本因素；
      2. 确定被评估设备的尚可使用寿命，计算每年的超额运营成本；
      3. 计算净超额运营成本（考虑税）；
      4. 确定折现率，计算超额运营成本的折现值。

    经济性贬值：是外部因素引起的贬值。

    1. 使用寿命缩短；
    2. 运营费用增加；
    3. 市场竞争加剧：贬值额或贬值率类比法一指数估价法间接获得。

    2. 市场法

    基本概念：前提条件：1. 公开市场、2. 市场有效、3. 相似或可比

    评估步骤： 1. 勘察。 2. 市场调查选取参照物。3. 确定比较因素，差异调整。 4. 计算评估值

    比较因素：个别因素、交易、时间、地域

    1. 个别因素：名称、型号规格、生产能力、制造厂家、技术指标、附件、出厂日期等
    2. 交易因素：交易动机、背景、数量等对价格会有影响
    3. 时间因素：选择与评估基准日最接近的交易案例
    4. 地域因素：应尽可能与评估对象在同一地区

    具体方法：直接比较、相似比较、比率估价

    1. 直接比较法：根据与评估对象基本相同的市场参照物，通过直接比较来确定评估对象价值的评估方法。
    2. 相似比较法：将与评估对象相似的市场参照物作为评估的基础，通过比较、调整评估对象与市场参照物之间的因素差异确定评估对象价值的评估方法。
    3. 比率估价法：在市场上无法找到基本相同或相似的参照物时，利用从大量市场交易中统计分析的同类型设备使用年限与售价的关系，确定评估对象价值的评估方法。评估值＝ 全新设备的价格 × 变现系数

    3. 收益法

    前提条件：1.要能够确定机器设备的获利能力，如净利润或净现金流量；2.能够确定合理的折现率。

    不具备独立获利能力的单项设备一般不用收益法。对于具有独立获利能力的生产线、成套设备等可用收益法。

    使用资产基础法评估整体企业价值时，收益法也作为一种补充方法，来判断设备是否存在功能性和经济性贬值。

    第4章：不动产评估23

    概述

    基本概念：

    1. 不动产：土地、建筑物及其他附着于土地上的定着物，包括物质实体及其相关权益。不包括海域、林木等。
    2. 不动产实物和权益
      1. 实物：土地、有建筑的土地、无土地的建筑、土地和建筑
      2. 不动产权益：本质是一种财产权
        1. 物权：所有权、用益特权、担保特权
        2. 债权：租赁权

    不动产清查核实

    目的：应全面了解不动产实物状况、权益状况和区位状况，掌握评估对象的主要特征。

    内容：不动产清查核实的内容有实物状态、权利状态、区位和交易状况、会计核算情况等四个方面。

    方法：逐项调查、抽样调查

    手段：通常可以采用目测、询问、查阅档案、专业机构鉴定意见等方式。

    需要收集的资料：1.评估对象的基本情况，如产权、平面图、照片等； 2.有关评估对象所在地段的环境因素和区域因素资料； 3.与评估对象有关的不动产市场资料，如市场供需状况、建造成本、租售价格等； 4.国家和地方涉及不动产评估的政策、法规和定额指标。

    1. 市场法

    评估操作步骤：1. 收集交易实例。2. 确定可比交易实例。3. 建立比较基础。4. 进行比较修正（对交易情况、交易日期、不动产状况等进行修正）。5. 计算比准价格，确定不动产价值。

    收集交易实例：运用市场法评估不动产价值，必须有一定数量的交易实例，这是市场法运用的基础和前提条件。

    选取可比交易实例：1. 与评估对象类似的不动产。2. 交易类型与评估目的吻合。3. 成交日期与评估基准日相近， 不宜超过1年，且不得超过2年。4. 成交价格为正常价格或可修正为正常价格。

    建立比较基础：统一资产范围、付款方式、税费负担、计价单位

    对交易实例价格进行因素修正：交易情况、日期、权益状况、区位、实物状况 1. 交易情况修正 2. 交易日期修正 3. 权益状况修正：$y=\frac{1-1/(1+r)^m}{1-1/(1+r)^n}$ 4. 区位因素修正：直接比较、间接比较 5. 实物状况修正

    求取比准价格 - 主要有简单算术平均法和加权算术平均法两种。 - 评估实践中，在进行因素修正时，单项修正幅度一般不超过20%，综合修正幅度一般不超过30%；修正后的可比实例价格最高价与最低价之比不应大于1.2。

    2. 收益法：r用市场提取法、安全利率加风险调整、各种投资收益排序插入

    概述：适用于经营性且有稳定收益的不动产价值评估，如商场、写字楼、旅馆、公寓等，对于政府机关、学 校、公园等非经营性不动产价值评估大多不适用。

    评估参数与确定 1. 收益期限：如果是土地与建筑物合成体，熟短原则。 2. 年总收入：总收入＝潜在毛租金收入＋其他收入－空置和收租损失 3. 年总费用：剔除不正常的费用支出，选择正常支出作为费用。 用来求取不动产净收益的不动产总费用不包含折旧费，通常包括管理费、维修费、保险费、房产税等。 4. 折现率和资本化率：资本化率：在直接资本化法中采用，是一步就将预期收益转换为价值的比率。通常用未来第一年净收益除以价格来计算。 1. 市场提取法：要求市场发育比较充分、交易案例比较多。 2. 安全利率加风险调整值法：通常选择银行中长期利率作为安全利率，考虑各种风险后，在安全利率基础上加风险调整值。简便易行，对市场要求不高，应用广泛，主观性强，不易掌握。 3. 各种投资收益率排序插入法：将收集的各种投资收益率大小排队，再判断被估不动产在该风险排序中的位置， 然后根据风险排序结合相邻投资的收益率确定被估不动产资本化率的大小。

    3. 成本法

    概述 1. 基本思路： 1. 房屋建筑物：以重置或重建可以产生同等效用的房屋建筑物所需投入的各项费用之和为依据，再加上一定的利润和应纳税金，并扣除贬值来确定待估不动产价值。 2. 土地使用权：也称为成本逼近法，是以取得和开发土地所耗费的各项费用之和为主要依据， 再加上一定的利润、利息、应缴纳的税金和土地所有权收益来确定土地使用权 价值的评估方法。 2. 适用范围：适用于不动产市场发育不成熟，成交实例不多，无法利用市场法、收益法等方法进行评估的情况。对于既无收益又很少有交易情况的特殊性的不动产评估比较适用。 3. 前提条件：持续使用、有历史资料、工程量可计量、具有损耗特性

    单独的土地使用权：成本逼近法。

    土地使用权价值 ＝土地取得费用＋土地开发费用＋税费＋利润＋利息＋土地增值收益 1. 土地取得费用估算 2. 土地开发费用估算： 1. 宗地红线外：几通 2. 宗地红线内：平整 3. 税费计算：一般包括耕地占用税、耕地开垦费、新菜地开发建设基金、征地管理费、教育费附加等。 4. 利息计算：计息期应为整个开发期和销售期。若土地开发费是均匀投入，则计息期为开发期的一半。 5. 利润计算：计算的关键是确定利润率或投资回报率。利润率计算的基数可以是土地取得费用和土地开发费用，也可以是开发后土地的地价，计算时，要注意所用利润率的内涵。 6. 确定土地增值 7. 修正确定土地使用权价值

    单独房屋建筑物：重置成本法

    1. 确定重置成本：＝建安综合造价＋前期费用及其他费用＋利息＋合理利润

      3. 管理费用
      4. 销售费用
      5. 利息
      6. 确定合理利润
      7. 销售税费：
        1. 销售税费 ＝重置成本/（1＋增值税税率）×{增值税税率×（1＋城市维护建设税税率＋教育费附加率 ＋地方教育费附加率）＋印花税税率}
        2. 重置成本＝建安综合造价＋前期费用及其他费用＋管理费用＋销售费用＋投资利息＋合理利润＋销售税费

    2. 确定成新率

      2. 一般采用年限法成新率和打分法成新率，再通过加权平均综合确定。

    4. 剩余法

    基本思路 - 又称假设开发法、倒算法，是将待估不动产开发后的预期价值，扣除正常投入费用、正常税金及合理利润后，依据该剩余值测算被估不动产价值的方法。 - 结果的合理性取决于：1.合理选择最佳开发利用方式；2.合理预测未来开发完成不动产市场价格。

    适用范围：1. 待开发不动产的评估（假设开发）。 2. 对已建成不动产中的房屋或土地的评估。 3. 将生地开发成熟地的土地评估，用开发完成后的熟地价减去土地开发费用。 4. 待拆迁改造的再开发地产的评估，这时的建筑安装费用还应包括拆迁费用。

    前提条件：1. 不动产开发必须有已得到相关规划部门批准，且在有效期内的明确的规划。 2. 假设土地或不动产的利用方式为最佳开发利用方式。 3. 成本与价格的测算必须符合合法原则，与当地不动产市场的实际情况吻合。

    剩余法公式 - 基本公式：待开发不动产价值＝预期开发完成后的不动产总价值－待开发不动产取得税费－续建开发成本－续建管理费用－销售费用－续建投资利息－续建投资利润－销售税费 - 各阶段的公式

      2. 是森林资源资产评估中最基本的方法。
    2. 市场成交价比较法：
      1. 对同一评估对象应选取三个及三个以上参照交易案例，对各估算结果进行分析判断后，可采用简单算术平均法、加权算术平均法、中位数法、众数法、综合分析法等方法确定评估结论，
      2. 并在评估报告中披露所采用的方法和理由。
    3. 收获现值法：理论上可用于任何年龄阶段的林木资产评估，但实际应用中一般用于中龄林和近熟林。
      1. 此种方法主要针对造林年代已久，用成本法容易产生偏差，离主伐期尚早，不宜采用市场法而提出的。
      2. 考虑因素：
        1. 林分主伐时的净收益；
        2. 间伐的纯收入；
        3. 投资收益率，一般采用平均收益率进行测算；
        4. 评估基准日到主伐时的营林成本；
        5. 林分质量调整系数
    4. 年金资本化法：是在永续经营条件下，将被评估森林资源资产年稳定收益作为资本投资的收益，按适当的投资收益率折算林木资源资产价值的方法。主要用于评估年纯收益稳定的森林资源资产。投资收益率必须是不含通货膨胀率的当地该类资产投资的平均收益率
    5. 周期收益资本化法：以实现森林资源永续利用为前提条件，适用于异龄林林木资源资产评估。
      • 刚择伐完：将终值除以终值系数得到年金，然后用年金除以资产化率得到价值。
      • 择伐完n年：比刚择伐完要乘以一个（1+i）的n次方系数（复利终值系数）
      • $$\displaylines{年金终值系数=\frac{(1+i)^n-1}{i} \年金现值系数=\frac{1-(1+i)^{-n}}{i}}$$
    6. 重置成本法：重置成本法最适用于幼龄林林木资产的评估。
      1. 注意事项：1. 运用重置成本法评估林木资源资产必须确定合理的投资收益率；2. 运用重置成本法评估林木资产不需要考虑成新率问题；3. 运用重置成本法评估林木资产必须根据林分质量调整估算评估值。 各块林分的价值必须用林分质量调整系数进行调整。
      2. 株数调整系数：最大为1，小于40%为造林失败等于0
      3. 树高调整系数

    经济林

    评估方法 产前期：重置成本法。市场完善也可用市场成交价比较法 初产期：重置成本法、收益法或市场成交价比较法 盛产期：收益法或市场成交价比较法 衰产期：木材市场价倒算法，剩余价值法

    产前期 1. 重置成本法：主要是成本费用投入，第一年的投入最多，以后各年较为一致。 2. 市场成交价比较法

    初产期 1. 重置成本法：初产期的经济林资产评估中重置成本的全价只计算到经济林资产年经营收入大于年经营投入的前一年， 2. 收益现值法：当初产期的经济林资产年经营收入大于年经营投入时，宜采用收益现值法进行评估。 3. 市场成交价比较法

    盛产林 1. 收益现值法：初产期阶段采用收益现值法应明确该品种经济林的经济寿命、拟评估经济林初产期和盛产期的平均产量，并分段计算。 2. 市场成交价比较法：经济林产期市场成交价比较法中林分质量调整要考虑成新率

    衰产期： 1. 衰产期经济林的产量明显下降，继续经营收益低，甚至出现亏损，应及时更新。这个阶段的经济林资产可用剩余价值法进行评估。 2. 特别是乔木树种的经济林中，其剩余价值主要是木材的价值，评估方法可用木材市场价倒算法。

    竹林

    新造未投产的竹林：适宜用重置成本法，也可用市场成交价比较法（参见林木资产评估）

    已投产年龄结构不合理：收益现值法

    已投产年龄结构合理：年金资本化法，市场成交价比较法

    花年竹林：年年都出笋长竹，年度间产量差异不大的毛竹林。

    大小年竹林：用周期收益资本化法计算现值

    林地资源资产评估

    影响林地资源资产评估价值的因素有立地等级、地利等级、森林经营的方式及强度、林产品的销售价格、林产品生产周期及利率、有林地与无林地差别、林地用途、林地交易市场是否活跃等。

    评估方法：林地资源资产的价值以林地上所获得的收益来体现。 1. 收益法 2. 市场法 3. 成本法

    1. 市场成交价比较法
      1. 林地立地质量差异修正：立地质量调整系数、地利等级调整系数
      2. 物价指数调整系数
    2. 林地期望价法：以实现森林永续利用为前提，并假定每个轮伐期林地上的收益相同，支出也相同，从无林地造林开始进行计算，将无穷多个轮伐期的净收益全部折为现值累加求和作为拟评估林地资产评估值。
    3. 年金资本化法：年金资本化法是以实现森林永续利用为前提，且林地每年有稳定的收益，按恰当的投资收益率求出林地资产价值的方法: E=A/p
    4. 使用期有限期林地评估方法：无限期乘个系数 $$1-\frac{1}{(1+p)^n}=1-(1+p)^{-n}$$
    5. 林地费用价法：林地费用价法是用取得林地所需要的费用和把林地维持到现在状态所需的费用来确定林地价格的方法。林地费用价法适用于林地购入后经改良使之适合于林业用途的林地评估。林地费用价由三种费用构成：购买林地及其他为取得林地所需的费用；林地取得后，为适于林木培育而投入林地的改良费用；从投入上述费用开始到评估基准日为止的年间费用的利息。购买土地和林地的维持或改良的费用一般按购买或支付的原价格，不用重置值，此时计算利息采用的利率为正常的商业利率，它包括通货膨胀率在内；如林地的购置费和各年的林地改良费均采用基准日的重置值，则其利率用不含通货膨胀的利率；

    异龄林林地：通常分割的方法是比例系数法和剩余价值法。 1. 比例系数法： 1. 周期收益资本化法计算的异龄林的收益现值按规定（约定或当地森林经营的习惯） 比例分为林地价值和林木价值两部分。 2. 考虑三个因素：1. 当地森林经营实践中习惯性的林价中山价（地租）所占份额。2. 林地的立地条件类型或地位指数级。3. 林地的地利系数。 2. 剩余价值法 ：将用周期收益资本化法计算的异龄林的收益现值，减去林地上现有林木价值作为林地的价值。

    森林景观资产评估

    森林公园的景观质量受山体、水体、气象等自然因素，以及交通便利程度、旅游适游期、区位条件、市场需求状况等经济地理因素的影响。 1. 市场成交价比较法 2. 年金资本化法：对市场发育比较成熟，年均收益相对稳定，景观资源开发、建设和管理已日趋完善的森林景观资产的 价值评估宜选择年金资本化法 3. 模拟开发法：此方法是森林景观资产评估中最常用的方法。 4. 重置成本法：森林景观资产的重置成本包括森林和旅游设施的重置成本。

    第6章：金融不良资产评估8

    概述

    定义：金融不良资产，是指银行持有的次级、可疑及损失类贷款，金融资产管理公司收购或者接管的金融不良债权，以及其他非银行金融机构持有的不良债权。 贷款人应当根据贷款风险状况将贷款分为正常、关注、次级、可疑和损失类。

    作用： 1. 为金融不良资产处置提供定价依据。 2. 为防范和化解金融风险提供专业支持。 3. 为考核金融企业经营业绩提供重要依据。 4. 此外还有突破：“提供贷款五级分类的判定、贷款质量分析、风险质量控制等服务”属于“金融机构经营管理中的价值判断和风险防范评估”，已经突破了原有的不良资产“处置”领域。

    清查核实

    业务类型： 1. 价值评估业务： 2. 价值分析业务：对无法履行必要资产评估程序的金融不良资产在基准日特定目的下的价值或者价值可实现程度进行分析、估算，并出具价值分析报告等咨询报告 3. 根据评估实践，确定执行价值评估业务或价值分析业务时，一般考虑的因素有： 1. 在约定时限内，是否能够充分获取评估对象的资料和信息。 2. 对评估对象的现场调查是否受到限制，如由于相关当事方不配合或其他原因，评估师无法进入现场实施清查、勘察等必要评估程序。 3. 对评估对象的法律权属资料和资料来源的查验是否受到限制。 4. 是否存在评估师认为对形成合理价值评估结论具有重要影响的其他事项。

    评估对象确定：债权资产、用以实现债权资产

    1. 债权资产：
      1. 不良贷款：银行持有的次级、可疑、损失类贷款
      2. 不良债权：包括逾期贷款、呆滞贷款及呆账贷款等，又可以分为资产管理公司收购或接管的金融不良债权和其他非银行金融机构持有的不良债权。
    2. 用以实现债权清偿权利的资产：
      1. 实物类资产：收购的以物抵贷资产、资产处置中收回的以物抵贷资产、受托管理的实物资产及其权益，以及其他实物资产。
      2. 股权类资产：商业性债转股、抵债股权、质押股权
      3. 其他资产：土地使用权、商标权、 其他经营权的收益权
      4. 债权资产：银行持有的次级、可疑及损失类贷款，金融资产管理公司收购或者接管的金融不良债权，以及其他非银行金融机构持有的不良债权

    清查核实方法：相关合同资料、现场调查

    1. 查阅分析相关合同及资料
      1. 查阅确认债权、抵押资产相关的义务主体
      2. 借助律师尽调
      3. 查阅证明债务关系的有关合同文件
    2. 现场调查：金额大的逐笔核实，金额小的随机抽查核实。

    金融不良资产评估方法

    价值评估方法：

    采用收益法、市场法等评估方法估算出债务企业价值后，通过市场类似债权在强制变现情况下的数据信息，统计分析确定债权回收折扣率，从而计算债权可回收价值。

    价值分析方法：关联方、债权本身

    1. 以关联方为分析对象：主要适用债务人或债务责任关联方主体资格存在、债务人或债务责任关联方配合。相关的义务主体包括：借款人、保证人、抵押人、质押人、贷款发放银行、承租人、优先权人、债务人的股东等。
      1. 假设清算法：
        1. 适用非持续经营或净现金流小而不稳，存在有效资产，且能够取得比较齐全的财务会计资料 ；
        2. 不适用企业资产庞大或分布广泛、不良债权占企业资产比例相对较小的项目
        3. 价格标准：强制清算价格、有序清算、续用清算
        4. 主要操作问题：
          1. 无效资产：福利性资产、待处理损失、待摊和递延资产
          2. 无效负债：长期挂账无须支付的负债、与福利性资产相对应的负债等
      2. 现金流偿债法：持续经营，财务规范，有稳定的现金流
        1. 现金流偿债法中折现率为基准利率(国债利率)与风险调整值之和
        2. 注意问题：
          1. 企业未来现金流量应包括预期偿债年限内由经营带来的现金流量以及预期偿债期末由资产变现带来的现金流量；
          2. 预期偿债年限、偿债系数、折现率的确定应当具有依据或合理解释；
          3. 在预测中应当分析抵押物对企业现金流的影响；
          4. 应当适当考虑企业非财务因素对偿债能力的影响，或在特别事项说明中予以披露
    2. 以债权本身为分析对象：
      1. 交易案例比较法：
        1. 一般程序①对债权资产进行定性分析。②选择交易案例。③在分析对象和参照物之间进行比较因素调整。④指标差异比较、量化。⑤合理分析估测债权资产价值。
        2. 局限性包括: 1. 案例的选取，由于市场上存在各种各样的案例，是否选取了同一或相似的资产类型、同一或相似的交易方式同一或相似的市场交易条件的案例，对评估结果影响较大。2. 案例的因素比较过程带有人为影响，对于不同因素孰重孰轻的认识是因人而异的，因此每个因索的修正程度也会不尽相同。
      2. 专家打分法：有增长潜质，持续经营或有效资产存在，然而企业不配合评估。通过匿名方式征询有关专家的意见。15人左右，经过3轮。
      3. 综合因素分析法：适用于存在诸多不确定因素、无法采用其他集中方法进行定量分析的债权。
        1. ①根据信贷档案资料确认抵押物，搜集、分析调查资料并查勘现场，经测算分析得出抵押物优先受偿债权价值。②对主债权人偿债能力从多个方面进行分析，得出一般受偿债权价值。③对担保人偿债能力从多个方面进行分析，得出保证受偿债权价值。④预计债权回收金额＝优先受偿债权价值＋—般受偿债权价值＋保证受偿债权价值。
        2. 运用中主要存在的问题：1. 影响偿债能力的评价指标设置不够全面。2. 指标价值影响系数设定未按照专家打分法的要求和程序进行论证、确定，缺乏依据。

    第7章：以财务报告为目的的评估10

    概述

    定义：所涉及各类资产和负债的公允价值或特定价值进行评定、 估算，并出具评估报告的专业服务行为。以财务报告为目的的评估是为会计的计量、核算及披露提供意见的一种专业服务。

    作用： 1. 评估技术能够满足会计计量专业上的需求。 2. 评估专业行为能够为会计计量的客观性奠定基础。 3. 评估的独立地位能够强化公允价值的公正性。

    对象：一般意义上，以财务报告为目的的评估业务是为编制财务报告服务的。

    特点： 1. 会计计量模式、会计核算方法、会计披露要求影响评估对象、价值类型的确定及评估方法的选择； 2. 以财务报告为目的的评估业务具有多样性、复杂性； 3. 具体使用的评估方法具有多样性。

    投资性房地产公允价值评估

    评估对象确定： 1. 已出租的土地使用权和已出租的建筑物，是指以经营租赁方式出租的土地使用权和建筑物。 2. 持有并准备增值后转让的土地使用权，是指企业取得的、准备增值后转让的土地使用权。

    基准日：应根据会计准则的要求合理确定评估基准日，可以是资产负债表日、投资性房地产转换日等。

    前提：投资性房地产的公允价值评估是基于资产的最佳用途产生经济利益的能力，或者将该资产出售能够用于最佳用途的其他市场参与者产生经济利益的能力的评估。

    市场法： 1. 选择类似房地产 2. 构建可比修正体系：一般从交易情况，交易日期和房地产状况选取比较指标。 3. 比较的方式：①分别对可比案例成交价格修正或者调整的幅度不宜超过20%，共同对可比案例成交价格修正或者调整的幅度不宜超过30%； ②经修正或者调整后的可比案例价格，最高值与最低值的比值不应超1.2； ③ 当修正幅度或者比值超过上述规定，应重新选择可比案例。

    收益法： 1. 确定收益期：孰短原则，并考虑期满残留价值。 2. 合理确定持有期：根据市场典型持有时间来确定，通常5-10年。 3. 预测未来收益：＝潜在毛收入－空置或者租金损失＋租赁保证金或押金利息－出租人负担的运营费用 4. 折现率：无风险报酬率＋风险调整值

    服务于金融工具计量的公允价值评估

    金融工具及计量：1. 分类：现金类、衍生类、2. 计量：初始、后续

    评估对象：1. 基础金融工具、2. 衍生：

    评估方法 1. 权益工具： 1. 从发行方看，权益工具通常指企业发行的普通股、在资本公积下核算的认股权等。 2. 可以根据实际情况分别采用收益法、市场法和成本法对权益工具的公允价值进行评估。 2. 不含衍生工具的金融负债：通常采用未来现金流量折现法确定。 1. 固定利率债券 2. 浮动利率金融负债的评估方法： 3. 可转换公司债券的价值评估方法有 BS 模型以及二叉树期权定价模型。 3. 金融衍生工具 1. 员工持股计划： 1. 与普通期权存在如下三点不同：企业与员工之间发生的交易；以获取员工服务为目的的交易；交易对价或其定价与企业自身未来价值密切相关。 2. 模型主要为 BS 模型和 Lattice 模型。 2. 互换合同：互换合同的公允价值实际上就是上述固定利率债券以及浮动利率债券公允价值的差额。 3. 嵌入衍生工具：常用于计量嵌入衍生工具公允价值的模型比较复杂，一般都采用 Lattice 模型进行评估。

    服务于企业合并对价分摊的资产评估

    概念： - 合并对价分摊是指符合企业合并准则的非同一控制下的企业合并的成本在取得的可辨认资产、 负债及或有负债之间的分配。 - 购买方对合并成本大于合并中取得的被购买方可辨认净资产公允价值份额的差额，应当确认为商誉。

    评估对象确定 - 合并对价分摊事项涉及的评估业务所对应的评估对象应当是合并中取得的被购买方各项可辨认资产、 负债及或有负债。可能确认的或有负债项目一般包括产品质量保证、不可撤销的亏损合同、未决诉讼、重组义务。 - 可辨认资产、负债的确认原则与识别：合并中取得的被购买方的无形资产或或有负债，其公允价值能够可靠计量的，应当单独确认为无形资产或负债，并按照公允价值计量。

    评估方法确定： 1. 有形资产和负债 2. 无形资产：存在活跃市场的，应按购买日的市场价格确定其公允价值；不存在活跃市场的，无法取得有关市场信息的，应按照一定的估值技术确定其公允价值。 3. 递延所得税的计算：不应折现。 4. 商誉的计算：公司管理层需关注其减值风险，并考虑及时执行商誉的减值测试程序。 5. 整体合理性测试：在合并对价分摊的评估中，以被购买方各项资产公允价值为权重计算的加权平均资本回报率，应该与其加权平均资本成本基本相等或接近。

    服务于资产减值测试的资产评估

    资产减值测试及流程

    资产减值测试对象： - 单项或组合。考虑：1.经营层面的独立性。2.合同的约束性限制。 - 资产减值测试评估对象应当与资产、资产组或资产组组合账面价值的成分保持一致。 - 除非不考虑该负债的金额就无法确定资产组的可收回金额，资产组的账面价值一般不应包括已确认的计息负债的账面价值。 - 营运资金： - 如果资产组或资产组组合账面构成中不包含营运资本，则评估对象也不包含营运资本。在此基础 上，预测期第一年的营运资本变动应在期初余额为零的基础上计算，即假设预测期第一年需要投入一笔额外的营运资本，并且到预测期结束时收回该笔营运资本投入。 - 如果资产组或资产组组合账面构成中包含营运资本，则评估对象也包含营运资本。预测期第一年的营运资本变动应以评估对象构成中包含的营运资本为期初余额进行计算。 - 总部资产 - 1.对于总部资产能够按照合理和一致的基础分摊至某资产组的部分，应当将总部资产的账面值分摊至该资产组。 - 2.对于总部资产难以分摊至任何资产组的部分，应当按照合理和一致的基础分摊至由若干个资产组组成的最小的资产组组合。

    价值类型选择：在资产存在减值迹象时，应当估计其可收回金额。可收回金额应当根据资产的公允价值减去处置费用后的净额和资产预计未来现金流量的现值二者之间较高者确定。

    评估方法选择 1. 公允减处置费：看是否存在活跃市场 2. 预计未来现金流：收益法

    评估参数确定 1. 资产预计未来现金流量的现值评估参数 1. 基本要求：以评估对象当前的状况为基础。不包括将来可能发生的、尚未做出承诺的重组事项或者与资产改良有关的预计未来现金流量。 2. 现金流预测期：一般只5年。 3. 资本性支出：不应当包括改良性资本支出和扩张性资本支出 4. 经济使用年限最后一年的净现金流量：需要考虑处置资产所收到或支出的净现金流，相当于预期公允价值减去处置费用后的净值，有可能为负值。 5. 所得税影响：应该基于税前基础 6. 折现率：与未来现金流匹配。 2. 公允价值减处置费用 1. 假设前提：持续经营假设，可以考虑将来可能发生的、 尚未做出承诺的重组事项或者资产改良；应同时考虑重组、改良对应的收益和成本、费用 2. 盈利预测：在各方面与市场参与者预期保持一致 3. 资本支出：可以包括扩张相关资本支出 4. 营运资金：维持现有企业运营并考虑企业扩张或重组所需的营运资金 5. 所得税：可以考虑为税后 3. 公允价值和使用价值评估模型，二者的主要差异体现为： 1. 收益期不同，分别为有限年期和无限年期； 2. 资本性支出的考虑不同； 3. 营运资金最终处理的方式不同； 4. 资产组收益期末是否有处置收益的考虑不同。
    注意：使用价值评估模型即资产预计未来现金流量的现值模型。

    第8章：其他资产评估5

    长期待摊费用评估

    概念：摊销期限在1年以上（不含1年，下同）的各项费用， 包括开办费、租入固定资产的改良支出以及摊销期在1年以上的固定资产大修理支出、股票发行费用等。

    特点：1.属于企业的长期资产；2.能使企业在以后会计期间受益。

    清查核实：1.抽查原始凭证（大额）、2.抽查受益期的有关文件，了解摊销政策、3.对该摊销数据进行复核计算

    评估程序：1.了解长期待摊费用核算情况、形成新资产和权利情况。2.选择合适的评估方法，一般可采用费用分摊法、账面余额法。 1. 费用分摊法：可以准确计算的某些预付性质或性质特殊的费用。如租入固定资产改良支出。 2. 账面余额法：对于尚存资产或者权利的价值难以准确计算的费用。如股票发行费。

    长期应收款

    概念 - 长期应收款指的是企业融资租赁产生的应收款项和采用递延方式分期收款、实质上具有融资性质的销售商品和提供劳务等经营活动产生的应收款项。 - 长期应收款既不是流动资产也不是固定资产，而是具有融资性质的金融资产，具体包括出租人融资租赁产生的应收租赁款，以及采用递延方式分期收款销售商品或提供劳务等经营活动产生的长期应收款。 - 该款项是据相关融资租赁合同或销售合同确定，并在一定的期限内按照一定的金额分期收取的租金或货款。

    清查核实： 1. 程序：1. 核对账证账表。2. 对长期应收款项进行减值测试，判断是否发生减值迹象。3. 估算可收回金额。 2. 方法：收集合同协议，函证，抽查，访谈，调查

    评估方法 - 长期应收款评估值＝账面余额－资产减值准备 - 减值客观证据： - 预计未来现金流量现值，应当按照该金融资产的原实际利率折现确定，原实际利率即初始确认该金融资产时确定的实际利率。

    生产性生物资产

    概念：为产出农产品、提供劳务或出租等目的而持有的生物资产，包括经济林、薪炭林、产畜和役畜等。

    特点： - 生产性生物资产能够在生产经营中长期、反复使用。 - 可分为未成熟阶段和成熟阶段。 未成熟阶段指尚未达到预定生产经营目的、还不能够多年连续稳定产出农产品、提供劳务或者出租的生产性生物资产；成熟阶段指已经到达预定生产经营目的的生产性生物资产。

    清查核实：1. 账证、账表、账实核对。 2. 抽样核实。3. 技术鉴定。

    1. 成本法：

      1. 基本思路：采用成本法评估生产性生物资产应该具备 4 个前提条件：
        1. 被评估资产的实体特征、内部结构及其功能必须与假设的重置全新资产具有可比性；
        2. 应当具备可利用的历史资料；
        3. 形成资产价值的各种耗费是必需的；
        4. 被评估资产必须是可以再生的或者说是可以复制的。
      2. 计算公式：评估价值＝基准日重置价值（相关成本）－有形损耗－无形损耗
        1. 外购资产
        2. 自行营造或繁殖
        3. 企业生产性生物资产收获的农、林产品成本，采用加权平均法、个别计价法、蓄积量比例法、 轮伐期年限法等方法，将其账面价值结转为农、林产品成本。
      3. 损耗：
        1. 有形磨损：主要是预计的产畜和役畜衰老及经济林老化等
        2. 无形磨损：预计因新品种的出现而使现有的生产性生物资产的产出能力和产出农产品的质量等方面相对下降、市场需求的变化使生产性生物资产产出的农产品相对过时等
      4. 局限性：误差大。价值的组成一部分是劳动投入，即人力，这同一般非生物资产相同；另一部分是自然增殖， 即自然生产力。成本法往往只考虑前者，而忽略了后者，通常会低估生物资产的价值。
      5. 适用：成本法适用于幼年期生物资产的价值评估。 对于无法利用市场法、收益法等方法进行评估的生物资产，理论上也可以选择成本法确定其价值。

    2. 市场法：运用市场法估算生产性生物资产的价值，关键在于参照物的选择及调整系数的设置。

      1. 适用条件：利用市场上同类或类似资产的近期交易价格，经过对比调整，确定被评估资产价值的一种评估方法。 运用市场法首先要有一个活跃的公开市场，再考虑公开市场上是否具有可比的资产及其交易活动。
      2. 计算公式：较一般非生物资产更加困难
      3. 局限性：
        1. 运用领域受到限制。 在生产性生物资产中，存在活跃交易市场的只有畜禽资产以及农作物资产中的花卉。对于大部分生产性生物资产而言，在现实中很难找到公平、活跃的交易市场，因此运用市场法将受到一定的限制。
        2. 调整系数难以确定。

    3. 收益法：

      1. 首要条件是分析生产性生物资产是否具有持续获利能力。 一般情况下，牧草、经济林资产、寿命期内可多次产出或繁殖的畜禽、亲鱼等都具有持续获利能力。
      2. 公式：折现
      3. 局限性：
        1. 难度大，主观性，不可预见性等
        2. 需要风险收益可靠准确量化才能用
        3. 尤其是折现率，必须选择同一市场上类似生物资产的平均报酬率。

    第1-1：无形资产评估概述3

    概念：特定主体拥有或者控制的，不具有实物形态，能持续发挥作用并且能带来经济利益的资源。

    内容：专利权、商标权、著作权、专有技术、销售网络、客户关系、特许经营权、合同权益、域名等可辨认无形资产和商誉这一不可辨认无形资产。

    特征： 1. 不具实物形态 、2. 能给其拥有者带来经济利益

    分类：

    1. 按取得方式：1. 自创无形资产。2. 外部取得：购入、接受投资、接收捐赠
    2. 按可否辨认：1. 可辨认。2. 不可：商誉。
    3. 按是否有专门法律保护：1. 有：专利权、著作权、商标权。2. 无：但受民法典等保护，技术信息、商业秘密
    4. 按性质和属性：1. 知识型：工业产权、著作权。2. 权利型：租赁权、特许经营权、专卖权。3. 关系型：客户关系、销售网络、专家网络、关系型无形资产是指特定主体通过提高企业经营管理水平、商品质量、服务质量和商业信誉等方面逐渐建立起来的经济资源。4. 组合型：商誉，如属于企业管理和企业文化范畴的无形资产。

    形式特征：非实体、排他、效益性、成本不完整、弱对应

    1. 非实体性：最显著特征，无实体性贬值
    2. 排他性
    3. 效益性：最本质特征，属于超额盈利
    4. 成本的不完整性
    5. 成本与价值弱对应性

    功能特征：依附、共益、积累、替代

    1. 依附性：依附于一定的实物载体
    2. 共益性：一项无形资产可以在不同的地点、同一时间、由不同的主体所使用。
    3. 积累性：无形资产的成熟程度、影响范围和获利能力处于不断积累和演进中。
    4. 替代性：这一特性也反映了其获利不稳定，会涉及预测无形资产的有效作用期间

    评估关注要点：具体属性、特定时间点

    1. 无形资产评估必须基于无形资产的具体属性：主要考虑的是经济属性中的价值属性。
    2. 确定无形资产的价值量必须基于特定时间点

    影响无形资产评估价值的因素有9个：无形资产的成本、机会成本、效益因素、使用期限、 技术成熟程度、转让内容因素、无形资产的发展趋势、更新换代情况和速度、市场供需状况、同行业同类无形资产的交易方式。

    评估特征

    1. 复杂性：1. 种类多可比性差、2. 大多用收益法，参数多、3. 与宏观环境密切相关
    2. 动态性和预测性
    3. 需要结合载体作用和空间
    4. 需结合法律保护状况
    5. 需结合所在行业性质
    6. 广泛应用收益法

    评估目的

    1. 出资：①专利资产 ②专有技术资产 ③商标资产 ④著作权资产等
    2. 交易：①所有权的转让 ②使用权的转让：独占使用权、排他使用权、普通使用权
    3. 质押：以质押为目的的无形资产评估选用市场价值作为价值类型，同时结合质押率进行无形资产价值确定。
    4. 法律诉讼
    5. 财务报告：涉及商誉减值测试、可辨认无形资产减值测试、合并对价分摊等。
    6. 税收：用于企业重组涉税、内部无形资产转移等情形。
    7. 保险
    8. 管理：①政府部门：基于行政事业单位资产管理、 国有资产保值增值等需要； ②企业：资产经营管理、实现价值提升等需要。
    9. 租赁

    评估对象界定标准：经济学标准、法律依据、市场标准

    1. 经济学标准：从无形资产目前和历史上的发展状况以及无形资产实施的地域范围、领域范围、 获利能力与获利方式，判断无形资产是否能带来显著、持续的可辨识经济利益。
    2. 法律依据：从委托人提供的法律文件、权属有效性文件或者其他证明资料，来判断无形资产存在、剩余经济寿命和法定寿命。
    3. 市场标准：从无形资产以往的评估及交易情况或相关无形资产的市场转让、出资、质押等情况，判断市场对被评估无形资产的认可程度。

    评估范围：单项、可辨资产组、其他资产组

    1. 单项
    2. 可辨认资产组
    3. 其他资产组：包含不同单项无形资产的种类和数量和不可辨认无形资产（商誉）的有关内容，同时也会涉及所依托的有形资产的种类、数量等具体内容。

    评估假设：基本、具体

    1. 基本假设：1. 持续使用：考虑无形资产自身是否尚有显著的剩余使用寿命。2. 公开市场：只有在公开市场假设的前提下，运用市场法才有意义。3. 清算假设：假设无形资产在非公开市场条件下被迫出售或快速变现。
    2. 具体假设：①与被评估无形资产相关的国家法律法规和政策在预测期无重大变化； ②国家现行银行信贷利率、外汇汇率无重大不可预见变化； ③国家目前的税收制度除社会公众已知变化外，无其他重大变化； ④被评估无形资产所在企业的经营模式、盈利模式在预测期内无重大变化； ⑤被评估单位会计政策与核算方法在评估基准日后无重大变化。

    第1-3：收益法在无形资产评估中的应用12

    1. 节省许可费用法

    评估思路： 1. 许可使用费包括一笔期初入门费和建立在每年经营业绩基础上的分成费。 2. 计算公式：两笔费用折现

    操作步骤 1. 确定入门费 2. 确定许可费率：1. 市场可比费率。2. 基于收益分成。 3. 确定许可期限：许可期限一般短于经济寿命年限，还可能短于法定保护期限。通常依据与被评估无形资产相同或相近无形资产在法律或合同、企业申请书中规定的许可使用期限，确定被评估无形资产的许可期限。 4. 确定折现率：可以采用风险累加法、回报率拆分法等方法测算。 5. 节省许可费折现

    注意事项 1. 许可费率的可获得性及可靠性 2. 适用情形 1. 节省许可费法须在可比资产存在、经济行为双方独立、熟悉情况并且自愿的情形下适用。 2. 节省许可费法多用于无形资产使用权转让、出租的评估，主要包括商标、专利以及技术特许。 3. 还可以用于无形资产侵权损失评价过程中提供参考依据。

    2. 增量收益法

    评估思路：预测使用该项无形资产的企业得到的利润或现金流量，与一个没有使用该项无形资产的企业所得到的利润或现金流量进行对比，将二者的差异作为被评估无形资产所创造的增量收益，然后再采用适当的折现率，将预测的每期的增量利润或现金流量转换成现值，或者运用一个资本化倍数，将恒定的增量利润或现金流量进行资本化，以得到无形资产的价值。

    操作步骤 1. 确定增量收益：与没有使用的对比，分为收入增长型和费用节约型 2. 确定收益期限：可以采用法定年限法、更新周期法以及剩余经济寿命预测法等具体方法 3. 确定折现率或资本化率：风险累加法、回报率拆分法 4. 增量收益折现

    注意事项： 1. 增量收益的合理性：避免多评少评 2. 适用情形：①可以使企业产生额外的现金流量或利润的无形资产（收入增长型） ②可以使企业获得成本节约的无形资产（费用节约型）

    3. 超额收益法

    评估思路： 1. 先测算无形资产与其他相关贡献资产共同创造的整体收益； 2. 在整体收益中扣除其他相关资产的贡献，将剩余收益确定为超额收益，作为目标无形资产所创造的收益； 3. 超额收益折现，或者运用一个资本化倍数，将恒定的超额收益进行资本化，获得无形资产价值。

    操作步骤 1. 确定超额收益：直接收益、间接收益、混合收益方式 2. 确定收益期：法定年限法、更新周期法以及剩余经济寿命预测法。与增量收益法相同 3. 确定折现率或资本化率：风险累加法、回报率拆分法等 4. 折现

    注意事项 1. 超额收益与组合收益：从组合收益中扣除其他资产相应贡献过程中，避免“多评”或“漏评”。 2. 可辨认与不可辨认产生的超额收益 1. 商誉：一旦脱离依附对象便失去了使用价值，此时应以其当前使用所产生的超额收益为基础进行评估 2. 可辨认：可适当考虑其更大范围内的使用价值，比如商标及通用性较强的技术型无形资产等。 3. 适用情况：①特许经营权、公路收费权、矿权等无形资产的评估中特别适用。 ②企业合并对价分摊、商誉减值测试、可辨认无形资产减值测试等以财务报告为目的的无形资产评估。

    增量收益与超额收益 1. 增量收益＝使用无形资产的收益－未使用无形资产的收益 2. 超额收益＝整体收益－其他相关贡献资产的相应贡献

    适用范围对比 1. 节省许可费：①无形资产使用权转让、出租的评估，包括商标、专利以及技术特许。 ②用于无形资产侵权损失评价过程中提供参考依据。 ③组成“权利束”的部分或许可权利。 2. 增量收益法：①可以使企业产生额外的现金流量或利润的无形资产（收入增长型）②可以使企业获得成本节约的无形资产（费用节约型） 3. 超额收益法：①特许经营权、公路收费权、矿权等无形资产的评估中特别适用。 ②企业合并对价分摊、商誉减值测试、可辨认无形资产减值测试等以财务报告为目的的无形资产评估。

    收益期限确定

    无形资产的收益期限：法定、合同、经济寿命、其他

    1. 法定寿命
    2. 合同有效期限
    3. 经济寿命
    4. 其他类型寿命：司法寿命、技术寿命、功能寿命

    无形资产收益期的确定原则

    1. 无形资产价值降低原因：被替代、非独有、需求下降快
    2. 确定原则：
      1. 法律或合同、企业申请书分别规定法定有效期限和受益年限的，按孰短的原则确定。
      2. 法律未规定有效期，企业合同或企业申请书中规定有受益年限的，可按照规定的受益年限确定。
      3. 法律和企业合同或申请书均未规定法定有效期限和受益年限的，按预计收益期限确定。预计收益期限可以采用统计分析或与同类资产比较得出。

    收益期确定方法：法定、更新周期、剩余经济寿命

    1. 法定年限法
    2. 更新周期法：产品更新周期（高科技行业）、技术更新周期
    3. 剩余经济寿命预测法：应与技术专家、市场营销专家进行咨询相结合，并据企业特征对个别因素进行修正。

    收益额的确定

    增量收益估算：收入增长、费用节省

    1. 收入增长型：1. 单价增高、2. 销量增加。
    2. 费用节约型：1. 在不影响产品质量和功能的前提下，减少人工、机械、材料的成本，比如提高每单位劳动投入的产出、降低机器保养费用和缩短检修的时间、使用更低成本的原材料。2. 提高产品质量，减少产品召回、废品及次品。 3. 提高经营效率，比如简化生产程序、降低资产维护费用。 4. 降低管理费用、广告费用、销售及促销费用、坏账支出等。 5. 减少或消除环境污染。
    3. 与行业平均水平比较：无形资产增量收益额＝企业收益额－净资产总额×行业平均收益率

    超额收益估算：超额收益＝企业的全部收益－归属于企业有形资产带来的收益

    收益分成估算：约当投资、经验数据、要素贡献

    1. 影响因素：①被评估无形资产的具体类型； ②被评估无形资产所在企业的经营类型； ③被评估无形资产所在企业的行业状况； ④被评估无形资产所在企业的实际经营能力分析； ⑤被评估无形资产所在企业其他资产为创造收益所做贡献的相对重要性分析； ⑥市场上相同或类似无形资产转让协议中已知的分成情况； ⑦分析中选择的测算经济收益指标应该以被评估无形资产所在企业其他有形及无形资产的公允价值为基础。
    2. 类型：销售利润分成率，销售收入分成率（常用）。
    3. 常用分成率测算方法
      1. 约当投资分成法：确认无形资产的约当投资量
        1. 无形资产的约当投资量 ＝无形资产的重置成本×（1＋适用成本利润率）
        2. 购买方的约当投资量 ＝购买方投入总资产的重置成本×（1 ＋适用成本利润率）
        3. 计算无形资产利润分成率。
      2. 经验数据法：三分法（资金，劳动力，技术）， 四分法（资金、劳动力、技术、管理）
      3. 要素贡献法：经验数据法的一种特殊表现形式。资金、技术、管理

    折现率的确定

    无形资产运营中的风险：宏观、微观

    1. 宏观：政策、法律、市场、技术
    2. 微观：研发、资产管理、财务、商业化

    折现率测算方法：风险累加、回报率拆分

    1. 风险累加法：无形资产折现率＝无风险报酬率＋风险报酬率
    2. 回报率拆分法（逆向倒推思路）：税后口径

    注意事项： 1. 无形资产评估中的折现率一般高于有形资产评估中的折现率 2. 无形资产评估中的折现率有别于企业价值评估中的折现率 3. 折现率口径应与收益额口径保持一致

    第1-4：市场法和成本法在无形资产评估中的应用9

    市场法应用

    评估技术思路

    1. 总价计量方式：如一项专利资产转让价 500 万元；
    2. 从价计量方式：收分成率、利润分成率等。如按照专利产品生产经营利润 30%的费用
    3. 总价与从价相结合的计量方式，即“入门费＋分成”的计量方式。这种计价方式实际是采用总价计算方式测算一个入门费，然后再外加一个从价计量方式的分成。

    总价计量方式技术思路：本身和相关业务资产组

    1. 可比对象选择：1. 无形资产的可比条件：功效、权利状态、发展阶段、2. 相关业务资产组可比条件：相同或相近业务、大小规模相同或相似
    2. 差异分析调整：1. 无形资产本身差异调整：主要是所处发展阶段、2. 相关资产组差异调整：规模调整、产品毛利差调整、维护支出成本差调整

    从价计量方式技术思路：收益贡献差异、入门费对分成影响差异

    1. 可比对象选择：1. 功效相同或相似、2. 对于采用“入门费＋分成”计量方式的，需要合理地分别测算入门费和分成率，或者采用一种合理的方式将入门费换算成分成，或将分成换算为总价计量的入门费。3. 实务中“入门费”经常可能是根据无形资产的转让方转让无形资产过程中所需要的成本估算。
    2. 差异调整：1. 无形资产对各自产品收益贡献的差异调整、2. 入门费绝对值大小对分成影响的差异调整。

    主要参数确定：对比公司法、交易案例

    1. 采用对比公司法测算分成率：
      1. 国内上市公司中寻找若干家与被评估无形资产拟实施企业处于同行业的公司作为“对比公司”；
      2. 计算对比公司无形资产分成率并计算其平均值；
      3. 对比公司无形资产分成率进行相关因素的调整后，测算出被评估企业标的无形资产的分成率；
      4. 应用：如：技术类无形资产的调整因素一般是销售利润率。
    2. 采用市场交易案例测算分成率：通过获取市场许可费交易案例数据对分成率进行测算，如美国 ktMINE 的无形资产许可费数据库。

    注意事项 1. 总价计量方式的适用性问题：总价计量方式的市场法评估一般不具有适用性。 2. 从价计量方式与节省许可费法的关系：1. 从价计量方式下的无形资产评估结果是以一个相对比率（利润分成率/收入分成率）的形式表现， 因此可操作性更强；2. 可比对象可以通过市场案例获得或者采用逻辑推算过程推算获得；3. 无形资产的从价计量方式的市场法评估其实是节省许可费法中的一个步骤。 3. 调整分成率应该关注： 1. 在对分成率进行调整时需要特别关注分成率的实质是单位产品收益中应该分给无形资产的比率，这是一个分配比例指标而不是一个绝对价值指标； 2. 该分成率是针对无形资产贡献率的指标而不是针对无形资产相关的资产组的获利能力的指标； 3. 无形资产相关的资产组获利能力强并不一定代表无形资产的贡献率高，即并不一定代表无形资产的分成率会高，这两者之间没有必然联系。

    成本法的应用

    成本法技术思路：被评估资产的评估值＝重置成本－实体性贬值－功能性贬值－经济性贬值

    无形资产重置成本特征：不完整性、弱对应性、虚拟性（象征性）

    自创重置成本测算：重置核算法、倍加系数

    1. 重置核算法：重置成本＝直接成本＋间接成本＋资金成本＋合理利润。直接成本不能按现行消耗量计算而按实际消耗量计算。
    2. 倍加系数法：
      1. 被评估对象为一项专有技术，在该项技术的研制过程中消耗材料12万元，动力消耗22万元，支付科研人员工资25万元。评估专业人员经过市场调查论证，确定科研人员创造性劳动倍加系数为1.5，科研的平均风险系数为0.5，该项无形资产投资报酬率为20％，行业基 准收益率为30％，试采用倍加系数法测算该项专有技术的重置成本。
      2. 重置成本=【（12+22+25x1.5）/（1-0.5）】x（1+20%）

    外购重置成本：市价类比、价格指数

    1. 市价类比法
    2. 价格指数法：以无形资产的账面历史成本为依据，用价格指数进行调整，测算其重置成本。采用物价指数法对无形资产重置成本计算时，考虑无形资产涉及两类费用，包括物质消耗费用和人工消耗费用，分别与生产资料指数和生活资料物价指数相关度较高。这两者指数均应采用个别物价指数而非综合物价指数。

    无形资产贬值率

    1. 实体性：一般不适用
    2. 功能性：
    3. 经济性：
    4. 特定无形资产可能没有贬值因素，例如商标，随着知名度增加，反而会提升价值。

    贬值率测算：专家鉴定法、剩余经济寿命

    1. 专家鉴定法
    2. 剩余经济寿命预测法

    注意事项 1. 组合劳动力：①无形资产发挥作用的模式都是需要与其他资产组成业务资产组，在业务资产组中必然包括组合劳动 力； ②组合劳动力属于商誉的组成部分，因此属于无形资产范畴，不能单独交易与转让； ③组合劳动力可以通过成本法评估，其价值包括劳动力的招募成本和培训成本，两者均可以采用重置成本测算。 2. 改编权：如果评估仅是改编权，而不是全部著作权，成本法很难适用。

    价值类型： 1. 市场价值：自愿买方和自愿卖方在各自理性行事且未受任何强迫的情况下，评估对象在评估基准日进行正常公平交易的价值估计数额。 2. 市场价值以外的价值主要有：在用价值、投资价值、残余价值、持续经营价值、清算价值、保险价值、课税价值等。

    第1-5：专利资产评估6

    专利与专利权

    概念： 1. 发明专利：收到申请之日起，保护20年。发明一般分为产品发明和方法发明两类。 2. 实用新型：10年。实用新型具有如下特征：第一，实用新型是一种新的技术方案。第二，实用新型仅限于产品，不包括方法。第三，实用新型要求产品必须是具有固定的形状、构造的产品。 3. 外观设计：15年

    专利权产生的条件 1. 积极条件：新颖性、创造性、实用性（①属于技术课题的解决方案；②具有再现性，在工业上能够制造和使用的现实可能性；③具有有益性，能够产生有益的社会效益。） 2. 消极条件

    专利权的申请 1. 发明专利和实用新型：请求书、说明书及摘要、权利要求 2. 外观设计：请求书、该外观设计的照片或图片以及对该外观设计的简要说明等，并且应当写明使用该外观设计的产品及其所属的类别 3. 审查和批准：1. 发明专利：发明专利的审查分为三个阶段：初步审查（18个月）、早期公布、实质审查（3年内）。2. 实用新型或外观设计：初步审查制。

    专利资产

    专利资产概念：1. 能持续发挥作用、2. 能带来经济利益、3. 获利能力法律保护

    法律特征：1. 时效性、2. 地域性、3. 约束性

    技术特征：1. 技术公开性、 2. 技术可能不完整性、3. 技术可能存在不成熟性

    经济特征：1. 垄断收益、2. 收益不稳定、3. 研发成本不易界定、4. 之间可比性弱

    专利资产评估对象的界定：基本状况、有效性

    一、明确专利资产的基本状况 1. 专利种类及名称 2. 专利申请号或专利号 3. 专利的法律状态：专利所处的专利审批阶段、年费缴纳情况、专利权的终止、专利权的恢复、专利权的质押，以及是否涉及法律诉讼或者处于复审、宣告无效状态。 4. 专利权利要求书所记载的权利要求 5. 专利资产的权利形式：专利权资产的权利形式一般包括所有权和许可使用权等。 1. 所有权 2. 许可使用权 1. 授权许可：1. 独占许可：转让方也不得在合同规定范围内使用该专利和销售该专利生产的产品。2. 排他许可：排他使用权（独家使用权）是指转让方在合同规定的时间和地域范围内只把技术授予受让方使用，同时转让方自己保留使用权和该专利产品销售权，但不再将该技术转让给其他人使用。3. 一般许可 2. 特别许可：开放许可、强制许可 3. 具有再许可权的许可权 6. 专利申请日和专利授权日 7. 单项专利或专利资产组合

    二、核实专利权的有效性 1. 是否有效属实 2. 是否有专利性：即使是有效的实用新型专利，仍可能因不具备“三性”（新颖性、创造性和实用性），经过无效程序，丧失专利权。

    专利资产价值影响因素

    1. 专利权的法律状态
      1. 发明专利申请的审批程序：受理、初步审查、公布、实质审查以及授权五个阶段。
      2. 实用新型或者外观设计专利申请的审批程序：受理、初步审查和授权三个阶段。
    2. 专利权利要求内容与保护
      1. 专利的权利要求是指该项专利要求法律保护的核心内容，如果恰当，则可以有效保护专利的核心， 避免出现侵权的可能，因而增加专利应有的价值。
      2. 质量高的专利权利要求应该使得权利保护范围大且没有非必要技术特征，权利要求清晰有条理，对出现侵权产品也容易取证。
      3. 权利稳定性一般是指专利权授权后对抗无效请求的能力。发明专利由于经过实质审查，其稳定性就高；而实用新型和外观设计专利有可能没有进行实质性审查，因此稳定性就低。一般来说，稳定性高的专利的价值要高于稳定性低的专利的价值。专利权无效宣告的理由具体包括：不符合《专利法》规定的实质性要件；专利说明书、权利要求书的撰写不符合法律规定；对专利申请文件的修改不符合法律规定；被授予专利权的智力成果不属于可授予专利权的范围；有在先申请等。
    3. 专利的创新性和实用性：专利的实用性表现为两个方面，其一是可靠，其二是实施成本尽可能低。
    4. 专利产品的发展前景
    5. 专利的应用领域与保护力度：1. 专利的“长度”：专利的保护期，2. 专利的“宽度”：专利的应用领域和专利的保护力度；

    1. 共有专利资产的评估问题
    2. 专利资产实施中需要用其他专利资产或无形资产的问题
    3. 价值问题：仅经济价值
    4. 质押目的评估的操作要点
    5. 出资目的：一般为所有权

    第1-6：商标资产评估9

    商标与商标权

    商标 1. 构成分类：文字、图案、符号等 2. 作用分类：1. 商品商标、2. 服务商标、3. 集体商标：是指以团体、协会或者其他组织名义注册，供该组织成员在商事活动中使用，以表明使用者在该组织中的成员资格的标志；4. 证明商标：是指由对某种商品或者服务具有监督能力的组织所控制，而由该组织以外的单位或者个人使用于其商品或服务，用以证明该商品或服务的原产地、原料、制造方法、质量或者其 他特定品质的标志。如绿色食品标志。 3. 功能分类：1. 常用商标、2. 防御商标：为了防止他人侵犯而申请使用一系列与自己商标近似而又相互联系的商标；3. 联合商标：联合商标是将同一商标在不同商品上注册，阻止别人在其他商品上使用自己已经注册的商标；4. 扩展商标：是指在同一商标的基础上，进行一系列的扩展注册， 如注册汉字商标后，还注册英文的意译商标、音译商标。 4. 享誉程度：普通商标、驰名商标 5. 是否受法律保护：注册、非注册商标。

    商标权 1. 商标权是指商标经注册或被认定为驰名商标而获得法律保护，形成了排他使用等权利。 2. 因《驰名商标认定和保护规定》的存在，即注册和非注册的驰名商标都会受到法律保护。驰名商标本身就是一种商标权。

    商标注册条件 1. 积极条件：显著特征。1. 商标本身具有显著性、2. 通过长期的使用获得商标的显著性。 2. 消极条件

    商标权的取得 1. 注册申请原则：自愿原则、先申请原则、优先权原则 2. 申请文件 3. 审查与核准：九个月

    商标权的保护范围 1. 注册商标的专用权，以核准注册的商标和核定使用的商品为限。 2. 注册商标所有人无权任意改变商标的构成要素，也无权任意扩大商标的使用范围。

    侵犯商标权的表现形式 1. 商标相同 2. 商标近似或者商品类似商标相同的侵权行为 3. 销售侵犯注册商标专用权的商品的侵权行为 4. 伪造、擅自制造和销售注册商标标识。 5. 反向假冒。 6. 帮助侵权。 7. 给他人的注册商标专用权造成其他损害的侵权行为。

    商标资产：形式特征、价值特征、法律特征

    商标资产概念：①能够通过营销在消费者意识中形成独特的联想并产生经济利益； ②以法律保护的形式将商标标识作用所带来的经济利益赋予了商标权利人。

    商标资产特征 1. 形式特征：1. 通常为商品商标权和服务商标权，集体商标和证明商标都存在着商标使用不具有专有性和排他性。2. 通常为驰名商标。3. 可以是独立的或以商标权为核心的资产组合。 2. 价值特征：1. 相对依附性、2. 相关技术和管理支撑、3. 广告宣传的维持助推作用、4. 扩展能力 3. 法律特征：1. 时效性：10年可续。前12+6个月宽限。连续3年停止使用的，商标权失效。2. 地域性：在法律认可的一定地域范围内受到保护。3. 约束性：商标权只有在核定的商品上使用时才受法律保护。

    商标资产评估对象的界定

    商标资产评估对象是指受法律保护的注册商标权益，不能带来超额收益的商标权难以成为商标资产，也难以成为评估对象。商标资产可以是独立的商标权或以商标权为核心的资产组合。

    评估客体的界定：法律状态、是否驰名、盈利模式

    1. 核实法律状态：①商标是否已经注册； ②注册商标是否有效。
    2. 明确是否驰名商标
    3. 明确盈利模式：商标的盈利模式包括因驰名商标的使用而形成商品或服务价格溢价或因商标的使用致使商标商品或服 务的销量增加等。

    评估客体约束条件界定：明确评估目的、权利形式、范围

    1. 明确评估目的：转让、投资、质押融资、侵权赔偿、财务核算等
    2. 明确被评估商标权利形式：转让或许可。相同或者近似的商标，应当一并转让。
    3. 明确使用对象范围和空间范围：或服务的范围（一系列商品还是单一商品）以及在多大空间范围内使用（国内还是国际）

    商标资产价值影响因素

    1. 宏观经济状况
    2. 市场影响力：知名度、信誉度
    3. 声誉的维护
    4. 商标权所依附的商品：商标商品的获利能力是决定商标资产价值的根本性因素。1. 所处行业及前景、2. 商品生命周期、3. 市场占有率及竞争状况、4. 经营企业的素质和管理水平、5. 商品获得能力
    5. 法律状态：1. 不注册使用与注册使用并行，仅注册才能产生专用权、2. 商标权的无效宣告、撤销及注销
    6. 商标的使用方式
    7. 类似商标的交易情况：使用市场法进行商标价值评估时，可比交易案例及其交易情况对商标价值评估起决定性作用。

    商标资产评估的注意事项

    1. 评估对象的确定：单一还是组合
    2. 商标资产贡献的确定
    3. 商标注册人和使用者分离时商标权益主体贡献的确定

    第1-7：著作权资产评估9

    著作权

    著作权相关概念 - 著作权概念：也称版权，是指文学、艺术作品和科学作品的创作者依照法律规定对这些作品所享有的各项专有权利。 - 著作权内容 - 人身权：1. 发表权、2. 署名权、3. 修改权、4. 保护作品完整权 - 财产权：可转让、继承、交易。复制权、发行权、出租权、展览权、表演权、放映权、广播权、信息网络传播权、摄制权、改编权、翻译权、汇编、其他 - 著作权相关联产品：著作权的客体为作品。 - 与著作权有关的权利（传播者权利）：指作品传播者等因其在传播作品过程中所做出的创造性劳动、投资或其他贡献而被法律赋予的权利。

    著作权具有以下特征：①著作权因作品的创作而自动产生；②著作权突出对人身权的保护。

    著作权的客体为作品：作品的特征包括：①作品是一种智力成果；②作品具有独创性；③作品具有一定表现形式。

    著作权的权属 1. 合作作品：作者对各自创作的部分可以单独享有著作权，但行使著作权时不得侵犯合作作品整体的著作权。 2. 职务作品：一般职务作品（单位优先2年）、特殊职务作品 3. 委托作品 4. 演绎作品 5. 视听作品 6. 原件所有权转移时的著作权归属 7. 著作权的集体管理

    著作权的保护：1. 合理使用。2. 法定许可

    著作权资产

    概念：的著作权的财产权益和与著作权有关权利的财产权益。

    特征：形式特征、法律

    1. 形式特征：1. 与相关有形无形资产共同作用、2. 与演绎作品共同作用、3. 与著作权相关权共同作用
    2. 法律特征：1. 时效性：于作者死亡后第50年的12月31日；单位为创作完成后第50年的1231，视听为首次发表后的第50年，但50年内没发表的不再保护。2. 地域性：一般来说，著作权的地域性限制越小，著作权评估值也就越大。

    著作权资产评估对象的界定

    两个维度：元素为著作权价值评估对象——××作品××权。 1. 作品类别 2. 著作权财产权益类别

    许可： 1. 法定许可：①使用的作品是已经发表的作品；②使用必须符合《著作权法》规定的具体情形；③使用的过程中不得侵犯著作权人的精神权利，不得影响作品的正常使用。 2. 授权许可：1. 专有许可、2. 非专有许可、3. 其他形式许可

    著作权资产价值影响因素

    1. 宏观经济状况
    2. 市场需要状况
    3. 著作权所依托的作品
    4. 著作权资产的运营模式
    5. 著作权资产的法律状态：不论登记与否都受法律保护。

    著作权资产评估的注意事项

    1. 评估对象的识别和清晰披露：1. 可以从作品类别和财产权利类别两个维度来识别评估对象；2. 再根据经济行为目的，进一步明确第三个层次的界定。如确定评估对象是××作品的 A、B 两项财产权益的普通许可使用权。
    2. 作品获取收益的方式
    3. 收益获取期限：文化产品寿命期限较短。
    4. 著作权资产面临的风险：1. 文化消费具有主观性、易变动性、不稳定性的特征。2. 政府文化政策的不稳定性也会造成文化需求波动的不确定性、3. 文化产品重复利用的价值较低，投资所形成的资产具有很强的资产专用性
    5. 关注合理分割著作权资产贡献的收益：具体分割的方法有经验数据（如分成率、许可费率等统计数据）、专家打分、层次分析、模糊评判等。

    第2-1：企业价值评估概述4

    企业价值

    特点 1. 是一个整体概念 2. 受企业可存续期限影响 3. 企业价值的表现形式具有虚拟性

    企业价值与业务价值：业务价值加和等于企业价值。

    企业价值的影响因素：宏观、行业、企业

    1. 宏观环境因素
    2. 行业发展状况：行业经济特征、行业市场结构、行业生命周期、行业景气程度
    3. 企业发展状况：1. 企业层面、2. 资产层面

    企业价值评估

    评估对象载体：多种资产组成的资产综合体

    评估关键：整体获得能力。是一种整体性评估。

    价值表现形式 1. 获利能力强：有形资产最佳使用价值、可确指无形资产价值、不可确指无形资产价值（商誉） 2. 较强，没达到行业平均：有形资产的最佳使用价值、专有技术等可确指无形资产价值 3. 微利或轻度亏损： 有形资产的重置成本价值 4. 严重亏损：有形资产的破产清算价值

    评估目的

    1. 企业改制、2. 企业并购、3. 企业清算、4. 财务报告、5. 法律诉讼、6. 税收、7. 财务管理、8. 考核评价、9. 其他目的

    评估对象

    1. 企业整体权益
      1. 从资本的运用角度看： 企业整体权益价值＝企业的总资产价值－企业负债中的非付息债务价值
      2. 从资本的来源角度看： 企业整体权益价值＝股东全部权益价值＋企业的全部付息债务的价值
    2. 股东全部权益：企业整体权益价值与股东全部权益价值是包含与被包含的关系。
    3. 股东部分权益
      1. 两种计算途径：1. 直接评估得出股东部分权益价值，比如，采用股利折现模型求取少数股权的价值； 2. 先评估得出股东全部权益价值，再乘以持股比例或持股数量，并考虑必要的溢价或折价因素后得出股东部分权益价值。
      2. 股东部分权益价值＝股东全部权益价值×股权比例＋控制权溢价或－缺乏控制权折价
      3. 控制权一般具有价值。并非所有的少数股权都伴随着折价。拥有控制权或缺乏控制权不是非此即彼的关系，两者之间没有一个明显的界线，也不能仅仅根据持股比例是否大50%来判断是否具有控制权。
      4. 缺乏控制权折价＝1－1/（1＋控制权溢价）

    评估范围

    1. 评估范围的界定：1. 表内外、2. 经营和非经营、3. 自身产权和控制产权
    2. 企业各项资产负债的识别：1. 经营属性：经营性和非经营性、2. 配置属性：必备和溢余资产
    3. 企业资产配置和使用情况分析：1. 经营性资产、2. 非经营性资产、3. 经营性和非经营性负债

    价值类型：市场、投资、清算

    价值类型：1. 市场价值、2. 投资价值：特定投资者价值、3. 清算价值

    选择依据：目的、市场条件、自身条件

    1. 评估目的：收购者关注投资价值（针对特定投资者）。出售方关注市场价值
    2. 市场条件：公开市场条件，非公开市场条件
    3. 评估对象自身条件：持续经营：可能选择市场价值或投资价值； 经营面临倒闭：可能选择清算价值

    评估假设：基本、具体

    评估假设：基本假设和具体假设 基本假设：1. 交易假设。2. 公开市场假设。3. 持续经营假设：最常用的假设。4. 清算假设 具体假设：1. 企业外部环境假设。2. 内部环境。3. 评估人员获取资料和履行程序方面的假设

    持续经营假设 1. 最常用 2. 考虑因素：评估目的、能否满足市场需求、企业要素的功能和状态 3. 分类：1. 存量持续经营假设、2. 增量持续经营假设、 3. 并购融合持续经营假设

    具体假设： 1. 外部环境假设 2. 内部环境假设 3. 评估程序方面的假设

    第2-2：企业价值评估信息的收集和评估6

    评估中的信息：内部、外部

    种类：①被评估企业内部相关信息；②被评估企业外部相关信息。 内部： 外部：1. 宏观环境。2. 区域经济。3. 行业发展。4. 证券市场。5. 产权交易市场。6. 可比企业资料

    信息收集

    意义：1. 信息收集是企业价值评估的基础、2. 信息收集是解决信息不对称的要求

    来源：1. 公开、 2. 非公开

    原则：1. 可靠性。2. 相关性。3. 有效性。4. 客观性。5. 经济性：①一般宏观经济、产业政策和资本市场等可采用专业数据服务商的数据； ②细分市场、特殊专业技术、关键交易案例、关键参数等，可以采用自行访谈或聘请外部专家的方式取得。

    基本程序 1. 明确目标、制定计划。2. 实施信息收集。3. 补充和追踪信息。4. 信息整理与分类

    宏观环境分析

    常见要点：1.国家、地区有关企业经营的法律法规； 2.国家、地区经济形势和未来发展趋势； 3.有关财政与货币政策。

    1. 政治与法律因素：1. 直接性、2. 难以预测性、3. 不可逆转性、
    2. 宏观与区域经济因素
      1. 社会经济结构：1. 产业结构：最重要的、2. 分配结构、3. 交换结构、4. 消费结构、5. 技术结构
      2. 经济发展水平：1. 规模、2. 速度
      3. 宏观经济政策
        1. 目标：充分就业、价格稳定、经济增长、国际收支平衡
        2. 工具：
          1. 需要管理政策：包括财政政策和货币政策。 基本政策有实现充分就业政策和保证物价稳定政策两个方面。扩张性和紧缩性。
          2. 供给管理政策：收入政策、就业政策、经济增长政策
          3. 国际经济政策：国际收支平衡
      4. 经济运行状况：1. 先行指标、2. 主要国际经济先行指标、3. 同步指标、4. 经济景气调查分析指标、5. 区域经济运行指标
    3. 社会文化因素：1. 人口、2. 生活方式和消费趋势、3. 文化传统和价值观
    4. 技术因素
      1. 科技水平：社会科技水平是构成科技环境的首要因素。包括科技研究的领域、科技研究成果门类分布及先进程度、 科技成果的推广和应用三个方面。
      2. 科技力量：包括基础研究、应用研究、技术开发三项。
      3. 科技政策
      4. 科技体制

    行业分析：经济特性、市场结构、生命周期、景气度

    常见要点 1. 行业政策规定 2. 行业竞争情况 3. 行业发展有利和不利因素 4. 行业特有的经营模式和特征 5. 企业所在行业与上下游行业的关联性

    1. 行业经济特性
      1. 市场规模
      2. 市场增长率
      3. 生命周期阶段：分为初创、成长、成熟、衰退四个阶段
      4. 竞争范围：分为全球性、全国性、地方性三档
      5. 竞争状况：要受以下因素制约：行业内现有竞争者的规模、行业成长速度、固定成本水平、产能释放状况、退出障碍。
      6. 消费者状况
    2. 行业市场结构：完全竞争、垄断竞争、寡头垄断、完全垄断

    3. 行业生命周期
      1. 初创：三大风险：一是高投入、高成本引起的经营风险；二是需求较低、销售渠道不确定带来的市场风险；三是可能由财务困难引起的破产风险。
      2. 成长
      3. 成熟：生产技术和工艺逐渐成熟、市场需求比较稳定、利润较高、风险较小、行业增长速度放缓。
      4. 衰退：新产品和大量替代品的出现。原行业的市场需求开始逐渐萎缩，产品的销售量也开始下降。行业企业着手退出向其他成长性更强或利润更高的行业转移。
    4. 行业景气程度

    五力模型： 1. 供应商的议价能力、 2. 购买者的议价能力、 3. 潜在竞争者进入的能力：受以下几个因素的影响：一是规模经济效应；二是产品差异程度；三是进入资本门槛；四是原材料来源及产品销售渠道；五是政策限制因素。 4. 替代品的竞争能力 5. 行业内竞争者现有竞争能力：受以下因素制约：行业内现有竞争者的规模、行业成长速度、固定成本水平、产能释放状况、退出障碍。

    模型

    1. 股利折现模型

    股利及股利分配 1. 股利概念 2. 股利分配：1. 相关规定、2. 主要模式

    股利折现模型具体形式 1. 基本公式 2. 零增长模型 3. 固定增长模型：①股利增长率要和企业的利润指标增长率相一致。 ②要注意任何企业都很难维持一个高于经济平均增长水平的增长率的规律。 4. 两阶段增长模型 5. 三阶段增长模型

    注意事项 1. 应用条件：适用于无控制权股东部分权益价值评估 2. 非经营性资产、负债和溢余资产处理：较大时适当考虑 3. 股利增长率和股利支付率的关系：一般前低后高

    2. 股权自由现金流量模型

    股权自由现金流量 1. 概念 2. 计算：股权自由现金流=（税后净营业利润+折摊）-（资本支出+营运资金增加）-（税后利息费用+付息债务净偿还）。税后净经营利润=EBIT*（1-T)=净利润+税后利息费用

    具体形式 1. 基本公式 2. 资本化公式： 1. FCFE以不变增长率增长且收益期无限。 2. V=FCFE/资本化率。 3. 资本化率=R-g 3. 两阶段模型 4. 三阶段模型

    注意事项 1. 应用条件：1. 能够对FCFE做出预测、2. 能够合理量化FCFE预测值的风险 2. 折现率选择：1. 应用股权资本成本、2. 通常可用资本资产定价模型、套利定价模型、三因素模型、风险累加法进行计算 3. 永续价值计算：应在预测期最后一期的股权自由现金流量基础上进行标准化调整，消除一个或者多个偶然因素的影响。

    3. 企业自由现金流量模型

    概念：FCFF。也称为实体自由现金流量，为全部资本投资者共同支配的现金流量。全部资本提供者包括普通股股东、 优先股股东和付息债务的债权人。与企业自由现金流量相比，股权自由现金流量显得更为直观，股权自由现金流量能够提示企业关注资金链问题。

    计算： 1. 企业自由现金流量＝（净利润＋利息费用×（1－所得税税率）＋折旧及摊销））－（资本性支出＋ 营运资金增加） ＝（税后净营业利润＋折旧及摊销）－（资本性支出＋营运资金增加） 2. 企业自由现金流量＝股权自由现金流量＋债权现金流量 3. 企业自由现金流量＝经营活动产生的现金净流量－资本性支出

    具体形式 1. 基本公式 2. 资本化公式： 1. FCFF以不变增长率增长且收益期无限。 2. V=FCFE/资本化率。 3. 资本化率=WACC-g 3. 两阶段模型 4. 三阶段模型

    注意事项 1. 应用条件：能预测FCFF, WACC 2. 折现率选择：WACC, 债务资本成本采用税后口径。 3. 资本结构的确定。 4. 永续价值计算：永续期新投入资本的回报率最终会趋于资本成本的水平。 5. 付息债务价值的评估：前提条件：企业承担的债务利息与债务资本投资者期望的回报率是一致的。 此时，付息债务折现值的合计数等于评估基准日该债务资本的账面价值。

    4. 经济利润模型

    概念：EVA=税后净营业利润-资本成本。运用前提是企业在新项目上的投入资本回报率等于历史投入资本回报率。

    计算 1. 经济利润＝税后净营业利润－投入资本的成本 2. 经济利润＝净利润－股权资本成本 3. 投入资本回报率＝税后净营业利润÷投入资本（期初数）

    优势： 1. 在衡量企业经营绩效的能力方面，经济利润显著优于财务报表中的会计利润。会计利润是权益资本口径的利润指标，经济利润是全投资口径的利润指标。 2. 相比于企业自由现金流量，经济利润更适于作为企业绩效的评价指标。经济利润可以对短期（或一年）的企业绩效做出衡量，但自由现金流量通常不适于对短期（或一年）的企业绩效进行评价，因为短期（或一年）的自由现金流量不仅取决于当年的税后净营业利润和非现金费用，还受新增资本投入的影响。

    具体形式 1. 基本公式 2. 资本化公式 3. 两阶段模型 4. 三阶段模型

    注意事项 1. 应用条件：能准确计算基准日投入资本，合理估计各期EVA，能合理量化风险。 2. 折现率选择：应选择WACC

    模型的对比和选择

    股权现金流VS股利现金流 - 全部分配时，两个模型可能相同结果 - 利润分配不稳定，用股权现金流更可操作 - 股权模型适用于控制权，股利模型适用于小股东

    企业现金流VS股权现金流 - 两种都可以得出股东全部权益价值。 - 企业现金流不考虑债务，但计算WACC还是要考虑，所以方便程度差不多。 - 对银行保险证券公司评估时，优先用股权模型。

    EVA和现金流 - 两者应该是等价的 - EVA表达出更丰富的内涵

    收益法应用的操作步骤

    （一）确定评估思路和模型 （二）分析和调整历史财务报表 （三）确定和划分收益期 （四）预测未来收益 （五）确定折现率 （六）测算经营性资产及负债价值：经营性资产及负债价值＝预测期收益现值＋永续期收益现值 （七）识别和评估溢余资产、非经营性资产及负债 （八）得出评估结果

    未来收益预测

    收益的界定 1. 收益的形式 1. 按属性：净利润、自由现金流、EVA 2. 按享有主体： 1. 权益资本收益指标：股利、净利润、股权自由现金流量 2. 全投资资本收益指标：息前税后利润、企业自由现金流量、经济利润。 2. 收益形式的选择 1. 自由现金流比净利润更可靠和相关性 2. 折现过程的效率和效果 3. 与企业的适用性：如在银行、保险公司、证券公司的控股性产权变动业务中，通常选择股权自由现金流量，一般不宜选择企业自由现金流量作为其收益指标。

    收益范围 - 税收是不能拥有的收益：不论是流转税还是所得税都不能视为企业收益。 - 全投资收益范围不能遗漏：无论是营业收支、资产收支，还是投资收支，只要形成净现金流入量，就应纳入企业收益范围当中。

    收益调整 1. 财报编制基础：统一 2. 非经常性收入支出：需要调出。偶然性其他业务收入属于非经常性收入。 3. 非经营性资产负债和溢余性资产等：调出。评估专业人员应恰当考虑这些项目的影响，并采用合适的方法单独予以评估。

    收益预测步骤 1. 对影响因素进行分析 1. 外部因素：宏观和行业 2. 内部因素：历史、现在、未来 2. 对历史收益分析和调整 3. 对收益趋势进行总体分析判断：主要包括企业当前所处发展周期及未来的走势、企业未来收益进入稳定状态所需的时间以及进入稳定状态后的趋势等。 4. 具体预测：利润表、资产负债表

    收益期的确定：企业的收益期包括：有限年期和永续年期两种。对于大多数正常经营的企业，在没有信息证明其企业经营有年限的限制时，均适用于永续经营假设， 收益期限为无限期； 1. 法律规定 2. 协议章程影响 3. 主要资产使用年限 4. 所处生命周期及经营状况

    收益期的划分 1. 详细预测期：应该足够长。本质上长短不影响企业价值，只影响在两期的分布。 2. 稳定期：接近行业平均。1. 收入成本稳定、2. 资本结构稳定、3. 除维持外不再新增投资、4. 投资收益水平稳定、5. 风险水平稳定

    主要预测方法 1. 定性：1. 一般调查预测、2. 集体意见、3. 头脑风暴、4. 德尔菲、5. 因素分析、6. 对比类推：产品类推法、地区类推法和局部总体类推法。7. 主观概率 2. 定量：1. 时间序列（1. 平均预测法、2. 指数平滑、3. 趋势外推）。2. 回归分析

    企业未来收益的具体预测 1. 收益预测基础。1. 历年及当期收益对未来收益的影响、2. 新产权主体对未来收益的影响 2. 收益预测的假设条件：①国家的政治、经济等政策变化对企业未来收益的影响，除已经出台未实施的以外，通常假定其将不会对企业未来收益构成重大影响； ②不可抗拒的自然灾害或其他无法预期的突发事件，不作为企业收益的相关因素考虑； ③企业经营管理者的个人行为也不在预测企业收益时考虑。 3. 收益预测的表现形式 ：可以运用利润表或现金流量表的形式，但并不等于说企业未来收益预测就相当于企业利润表或现金流量表的编制。 4. 企业收益构成项目的预测：1. 营业收入预测、2. 成本费用、3. 资本性支出：存量资本性支出、增量资本性支出、4. 营运资金预测：综合分析、分项预测 5. 收益预测合理性分析：①将详细预测期企业收益的发展方向和变化趋势与企业历史收益进行对比。 ②对影响企业价值的敏感性因素进行严格检验。 ③对所预测的企业收入与成本费用变化的一致性进行检验。若企业未来的收入变化而成本费用并无变化，则该预测值不合理。 ④将收益法评估结果与其他方法的评估结果进行比较，进而对收益预测值的合理性做出检验。

    折现率的确定

    股权资本成本：资本资产定价、套利定价、三因素、风险累加法

    1. 资本资产定价模型 ：资本资产定价模型是估计股权资本成本时被最广泛使用的方法。
      1. 无风险报酬率：实务中，无风险报酬率通常选取与企业收益期相匹配的中长期国债的市场到期收益率。收益期在10年以上的企业选用距评估基准日10年的长期国债到期收益率。
      2. 贝塔系数：
        1. 周期性公司的β系数高于非周期性公司；
        2. 经营杠杆较高的公司的β 系数高于经营杠杆较低的公司；
        3. 财务杠杆较高的公司的β 系数也高于财务杠杆较低的公司。
        4. 计算：回归分析法、可比公司法
        5. 影响因素：市场整体水平的指数种类选择、观察间隔期选择（每周或每月）、样本期间长度选择（5年或更长）
        6. 先去杠杆再加杠杆：杠杆=（1+（1-T）XD/E），即（1+税后产权比率）。去杠杆时用可比公司贝塔除杠杆，加时用无杠杆贝塔乘杠杆。
      3. 市场风险溢价：
        1. 估算方法：1.面向未来的方法：贴现现金流量法。2. 基于当前的方法：回归分析，以预测市场风险溢价。3. 基于历史的方法：收益变现法。（常用）
        2. 选长一点，用几何平均，一般情况下，几何平均法得出的预期风险溢价比算术平均法要低一些。
      4. 特定风险调整系数：如果市场上存在足够多的可比上市公司，并且这些可比上市公司与被评估企业有类似的风险特性， 无须再对被评估企业进行特定风险调整。
    2. 套利定价模型：资本资产定价模型是其特例。目前应用不广泛。
    3. 三因素模型
      1. 三个因素：市场超额回报率、期望的规模风险溢价、期望的价值风险溢价。
      2. 比套利定价模型客观
    4. 风险累加法：
      1. 风险报酬率＝行业风险报酬率＋经营风险报酬率＋财务风险报酬率＋其他风险报酬率
      2. 粗略和主观。资本资产定价模型可理解为是在风险累加法基础上通过引入β系数进行扩展得到的模型。

    收益法的适用性和局限性

    适用性： ①被评估企业满足持续经营假设 、②被评估企业未来收益可预测、③所获取评估资料的充分性

    局限性：①受市场条件制约（如，β 的计算）、 ②受企业营运期影响 、③部分评估参数对评估结果的影响非常敏感（折现率、增长率）、 ④具有较强的主观性（收益期、收益额、折现率）

    第2-4：市场法在企业价值评估中的应用15

    市场法评估的基本模型

    市场法所依据的基本原理是市场替代原理 1. 上市公司比较法 2. 交易案例比较法

    应用原则：可比、可获得、及时、透明、有效

    1. 可比性原则：关键。包括行业可比、规模、成长预期、经营风险、财务风险可比等。
    2. 可获得性原则
    3. 及时性原则
    4. 透明度原则
    5. 有效性原则：建立在市场合理有效的假设基础上。

    市场法常用的两种具体方法

    上市公司比较法

    可比公司的选择： 1. 采用主观调整法：质量重于数量 2. 采用回归法调整：应保证可比公司数量

    价值比率的选择：分子分母匹配，数据口径一致。1. 基本因素法。2. 统计方法。3. 常规方法

    价值比率的调整：1. 主观调整。2. 矩阵法。3. 回归法

    缺乏流动性因素的影响：非上市公司对比上市公司 控制股权：缺少变现性 少数股权：缺少交易市场 研究：1. 新股发行价格估算方式。2.非上市公司并购市盈率与上市公司市盈率对比方式

    交易案例比较法

    交易情况差异调整：采用的可比公司一般为非上市公司，在流动性方面比较接近，一般不进行调整。 - 交易条款 - 交易方式 - 交易时间差异

    案例获取渠道 - 产权交易所 - 上市公司收购案例

    操作步骤

    1. 选择可比对象
      1. 业务相同或相似，并力求现金流、成长潜力和风险水平方面的相似，
      2. 从行业因素、规模因素、成长预期、经营风险、财务风险等角度加以考虑
      3. 在采用某些特定方法的情形下，被评估企业自身也可以作为可比企业。（VM 指数法）
    2. 统一财报编制基础
      1. 会计政策差异调整内容：①存货成本核算；②收入确认；③折旧差异；④税收差异；⑤其他差异。
      2. 特殊事项调整内容：①非经营性项目调整；②非经营性及溢余资产调整。
    3. 计算各价值比率
      1. 分子分母匹配
      2. 数据口径一致
      3. 时间跨度一致
      4. 计算方式一致
    4. 选取价值比率
      1. 以分母性质分类：盈利比率、资产比率、收入比率和其他。
      2. 以分子性质分类：权益价值比率及企业整体价值比率
    5. 进行比较
      1. 定性：swot
      2. 定量：财务经营状况分析等
    6. 对价值比率进行调整：主观调整、矩阵法、回归法
    7. 将调整后的比率用于被评估企业
    8. 综合考虑市场法评估结果的差异
      1. 服务性企业：基于收入评估价值
      2. 资本密集型企业：基于净利润或净资产
      3. 房地产企业：基于毛现金流评估价值等。
    9. 进行调整：例如控制权溢价

    可比对象选择

    一般关注要点：1. 行业性质或者经济影响因素、2. 注册地业务活动范围、3. 业务结构、4. 经营模式、5. 企业规模、6. 资产配置和使用情况、7. 所处经营阶段、8. 成长性、9. 经营风险、10. 财务风险

    上市公司选择特别关注：数据充分、交易活跃、注册地

    1. 股票交易历史数据充分性
    2. 股票交易活跃程度
    3. 企业注册地和业务活动地域范围。

    交易案例选择特别关注：交易日、资料可获得充分性

    1. 交易日期尽可能接近基准日
    2. 案例资料的可获得性和充分性

    价值比率的选择及确定

    价值比率分类 1. 按分子计算口径：1. 权益价值比率：PE、2. 企业整体价值比率：EV/EBIT 2. 按分母计算口径：盈利价值比率、资产价值比率、收入价值比率、其它 3. 考虑因素：1. 选择的价值比率有利于合理确定评估对象的价值； 2. 计算价值比率的数据口径保持一致； 3.应用价值比率时对可比对象和被评估企业间的差异进行合理调整。

    市盈率 - PE = DPS1/(r-g) - PE = b*(1+g)/(r-g), b为股利分配率

    市盈率相对盈利增长比率PEG - PEG=PE/年盈利增长率x100 - 适用于高增长行业

    EV/EBIDA - （股权价值+债权价值）/（息税折摊前利润） - 用于大量固定资产投资的资本密集型公司

    EV/EBIT - 当企业在税率、资本结构等方面有较大差异时

    市净率PB - PB=ROExPE - 有大量资产且净资产为正

    EV/TBVIC - （股权价值+债权价值）/（总投入资本资产价值）

    收入价值比率 - 销售净利率 x PE - 用于成本率较低的服务业或者利润水平稳定的传统行业。

    价值比率选择方法：基本因素法、统计方法、常规方法

    1. 基本因素方法：主要考虑运用与公司价值相关性最高的变量
    2. 统计方法：回归分析，其中相关性最高的就是可以进行最佳解释的价值比率。
    3. 常规方法：根据多年来的实践和总结，评估某些行业常用的价值比率。

    价值比率的调整内容：财务、规模、成长性、其他

    1. 财务绩效调整：盈利能力、运营能力、偿债能力、成长能力
    2. 规模及其他风险因素差异调整
    3. 成长性差异调整
    4. 其他风险因素调整
    5. 采用交易案例法时，还应当注意交易条款、交易方式、交易时间等方面的因素调整。

    价值比率的调整方式：因素调整、矩阵法、回归法

    1. 因素调整法：在比较各公司间价值比率之后，应该考虑影响价值比率的主要因素，并看看是否可以解释其中的差异。例如，当采用资产比率（P/B）比较时，就需要关注资本报酬率（ROE）与 P/B 之间的关系。
    2. 矩阵法：在矩阵方法中，可以把价值比率根据因变量分成 4 块。例如，应用资产比率（P/B），资产报酬率（ROE）将比率分块。
    3. 回归法：矩阵法的局限在于很容易识别极端值，但对于并没有过分高估或低估的公司不容易分辨。而 回归法则是区分公司间差异更具有说服力、更通用的方式。

    价值比率的确定：中位数、算平数、调和平均、变异系数

    1. 采用中位数
    2. 采用算术平均数
    3. 采用调和平均值：指的是倒数的算术平均值的倒数。
    4. 依据变异系数选择：标准差除均值

    价值比率的计算时间区间：时点型、区间

    • 时点型：评估基准日
    • 区间型：
      • 分子可以基准日前若干日的平均，考虑季报影响30或60日。
      • 分母以前12个月为基础。
      • 突发事件应调整

    市场法的适用性和局限性

    适用前提 - 市场有效性：1. 存在大量以追求利润最大化为目标的理性投资者，个别投资者不能单独对市场定价造成影响。2. 市场信息充分披露和均匀分布，投资者所获取的信息是对称的。3. 投资者获取信息不存在交易成本。4. 投资者对信息变化会做出全面、快速的反应，且这种反应又会导致市场定价的相应变化。 - 数据充分性 - 数据时效性

    局限性 - 资本市场波动较大 - 难以找到相同类似的可比对象 - 价值影响因素难以考虑周全 - 主观判断空间大

    资产基础法技术思路

    基本原理：股东全部权益价值=表内外各项资产价值-表内外各项负债价值

    操作步骤 1. 确定评估范围 2. 现场调查、资料收集、核查验证 3. 评估各项资产和负债 4. 评估结论的确定和分析

    评估范围确定 1. 评估范围与所经济行为的一致性 2. 重要资产负债的完整性 3. 资产负债表范围的可靠性：可要求委托人提供经独立第三方实施专项审计的资产负债表 4. 财报数据口径的恰当性：一般采用母公司口径的个别资产负债表

    现场调查 - 手段包括：询问、访谈、核对、监盘、勘查等。 - 根据重要性原则采用逐项或者抽样的方式进行现场调查。

    假设前提 1. 常用的评估假设 1. 最佳使用假设：法律允许、技术可能、经济可行。 2. 持续使用假设：现状使用、原地续用、异地使用 3. 清算假设：有序变现、强制变现 1. 评估假设的选择 1. 进行财务报告为目的：考虑相关会计准则规定 2. 某项资产的最佳用途可能与其单项资产评估时不一致 3. 各项资产对企业价值的贡献也与单项评估是不完全一样

    各项资产评估方法 1. 长投 1. 需要单独评估情形：具有控制权，或者重要 2. 可以不单独评估：无控制权，不重大。 2. 资产组评估

    利用审计报告 - 审计报告的时效性和可靠性 - 注意审计意见类型

    引用单项资产评估报告 1. 引用情形：事先约定，土地估价报告 2. 注意事项 1. 报告的获取与保存：工作底稿 2. 评估结论的引用

    合理性分析 - 对表外资产和负债的检验 - 常见可能表外资产： - 1. 有获得专利管理机关颁发证书的专利或专利申请； - 2.自创无形资产，该无形资产投入账面没有反映； - 3.企业毛利率明显高于同行业平均水平； - 4.存在某种形式的特许权利（有些特许权利法规可能不允许单独转让，但可以随企业权益一同转让）； - 5.企业持有较知名商标（被冠以驰名商标、著名商标等）； - 6.存在著作权； - 7.具有独特的经营模式； - 8.协议约定的企业获益形式，如优惠贷款利率、优厚供应条件等。 - 常见可能表外负债 - 1.法律明确规定的未来义务，如土地恢复、环保要求等； - 2.和其他经济体以协议形式明确约定的义务。 - 与其他评估方法初步结果比较分析 - 对持续经营企业，一般不宜只采用资产基础法一种方法评估 - 多种方法时，采用定性或定量分析方式形成结论 - 如不能确定某种方法的评估结果最合适，国外评估机构经常按乐观、最可能、悲观三种情形，对折现现金流以不同的折现率进行敏感性分析，并与市场途径的结果、资产途径结果一起进行统计分析。根据评估目的，选择中位数、下四分位数或者上四分位数作为评估区间，或选择单一评估结果。

    适用性和局限性

    适用前提 - 继续使用假设下进行 - 可取得充分的历史资料 - 无法确定盈利情况或参照对象 - 可能对象： - 1. 无形资产较少，特别是不存在商誉的企业 - 2. 可能进入清算状态的企业 - 3. 开发建设初期的企业

    局限性：在持续经营假设下，资产基础法一般不应当作为评估企业价值唯一使用的方法。 - 难以发掘某些无形资产对企业创造的价值 - 通常所用时间长，成本高 - 难以体现企业整体获得能力 - 对于一些高科技企业和服务性企业，资产基础法尤为不适用。
